### DRAM память в гетерогенных системах

GPU и CPU оба напрямую подсоединены к DRAM чипам. У каждого GPU свой чип памяти. Со стороны device code, DRAM, присоединенная к GPU называется *глобальной памятью* (*global memory*), так как доступна всем [[Модель железа GPU#^e22679|SM]]. DRAM, которая присоединена к CPU называется *системной памятью* (*system memory*). CPU и GPU имеют единое унифицированное виртуальное пространство памяти.

CUDA API позволяет выделять память на CPU, GPU и копировать ее меж ними. Локальность данных может быть явно контролироваться, когда нужно. Unified memory позволяет размещать память, которая автоматически обрабатывается рантаймом CUDA и т.д.

### On-chip memory на GPU

В дополнение к глобальной памяти, каждый GPU имеет некоторую память на чипе. Каждый *SM* имеет свой файл регистра и общую память. Эта память является частью *SM*, и потоки могут обратиться к ней с очень высокой скоростью. При этом эти потоки должны находиться в текущем *SM*, в других она не доступна.
