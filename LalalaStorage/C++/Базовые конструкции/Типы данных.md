### Области видимости

В C++ есть понятие области видимости. Эта область ограничивается блоком кода, в котором переменная определена.
```C++
#include <iostream>
int a = 1; // глобальная переменная
int main() { 
	int b = 2; // локальная переменная 
	{ 
		int c = 3; // локальная переменная внутри блока 
		std::cout << a << " " << b << " " << c << "\n"; // корректно 
	} // Эта строчка не скомпилируется, 
	  // так как переменная c не определена в данной области: 
	std::cout << c << "\n"; 
}
```

### Простые типы
```C++
int main() { 
	char c = '1'; // символ 
	bool b = true; // логическая переменная, принимает значения false и true 
	int i = 42; // целое число (занимает, как правило, 4 байта)
	short int si = 17; // короткое целое (занимает 2 байта) 
	long li = 12321321312; // длинное целое (как правило, 8 байт)
	long long lli = 12321321312; // длинное целое (как правило, 8 байт) 
	float f = 2.71828; // дробное число с плавающей запятой (4 байта) 
	double d = 3.141592; // дробное число двойной точности (8 байт) 
	long double ld = 1e15; // длинное дробное (как правило, 16 байт) 
}
```

Символы записываются в апострофах, а не как в строки, в кавычках. Фактически в памяти хранится код ASCII, который равен 49. 
Узнать размер переменной можно при помощи оператора `sizeof`. Размер переменной зависит от разрядности системы. Поэтому есть заголовочный файл `cstdint`, в котором есть целочисленные типы с фиксированном размере.

### Переполнение целочисленных типов

Стандартные числовые типы имеют фиксированные размеры и ограниченное множество допустимых значений. Поэтому в результате арифметических операций может произойти переполнение.
Беззнаковые типы можно спокойно переполнять: вычисления будут производиться по модулю соответствующей степени двойки.
Наоборот, для знаковых типов переполнение приводит к так называемому [неопределённому поведению](https://en.cppreference.com/w/cpp/language/ub) (UB, undefined behavior).
Такая ситуация не приведёт к ошибке компиляции, но при этом стандарт C++ уже не будет гарантировать какое-то поведение программы.

**Беззнаковые типы** следует использовать, когда вы имеете дело с битовыми наборами. В остальных случаях предпочтительнее использовать **знаковые типы**.

### Числа с плавающей точкой
В C++ есть 3 встроенных типа чисел с плавающей запятой: `float`, `double`и `long double`. Хранение чисел с плавающей точкой в С++ основано на стандарте IEEE 754. Число представляется в виде двоичной дроби в экспоненциальной записи: отдельно хранится бит знака, порядок и мантисса.
