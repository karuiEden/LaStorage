### Модель программирования

Модель программирования CUDA предполагает гетерогенную компьютерную систему, т.е. систему с CPU и GPU. CPU и память, которая напрямую к нему подключена, называются *host* и *host memory*, соответственно. А GPU и память, которая напрямую к нему подключена, *device* и *device memory*.

Приложения CUDA запускают часть кода на GPU, но стартуют они всегда на CPU. *Host code*, т.е. код, запущенный на CPU, может использовать CUDA API для работы с памятью, запуска кода на GPU и т.д.

Код приложения, который запускается на GPU, называется *device code*, а функция, вызванная для исполнения, называется *ядром*(*kernel*). А процесс запуска работы ядра - *запуском ядра*.
