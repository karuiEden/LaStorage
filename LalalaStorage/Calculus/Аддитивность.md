**Лемма:** Если $f\in \mathcal{R}[a,b]$, $[c,d]\subset[a,b]\implies f\in \mathcal{R}[c,d]$ 

**Лемма:** Если $c\in(a,b)$ и $f\in \mathcal{R}[a,c]\cap \mathcal{R}[c,b]\implies f\in \mathcal{R}[a,b]$
$$\int_{a}^{c}f(x)dx+\int_{c}^{b}f(x)dx=\int_{a}^{b}f(x)dx$$
*Док-во:* 
**Следствие:** $f\in \mathcal{R}[a,b]\implies \forall c\in[a,b]:$$$\int_{a}^{b}f(x)dx=\int_{a}^{c}f(x)dx+\int_{c}^{b}f(x)dx$$

$p=\{ x_{i} \}_{i=1}^{n}, \ q=\{ x_{i} \}_{i=1}^{n}\cup \{ c \}$ 

$$w(f,[x_{j-1},x_{j}])\leq2M(x_{j}-x_{j-1})<2M\delta$$
$f\in \mathcal{R}[a,c]\cap \mathcal{R}[c,b]\implies f\in B[a,c]\cup B[c,b]\implies f\in B[a,b]$. Пусть $|f|<M$ 

$$\Omega_{[a,b]}(f,p)\leq\Omega_{[a,c]}(f,p_{1})+2M\delta+\Omega_{[c,b]}(f,p_{2})< \frac{\varepsilon}{3} + \frac{\varepsilon}{3} + \frac{\varepsilon}{3}=\varepsilon\implies$$
$\implies$ при достаточно малом $\delta$ $\Omega_{[a,b]}(f,p)<\varepsilon\implies f\in \mathcal{R}[a,b]\implies$ Пусть $\{ p_{n} \}$  - последовательность разбиений, таких, что $c\in p_{n}$ 
$q_{n}=\{ x\in p_{n} | x<c \} \cup \{ c \}$ $r_{n}=\{ x\in p_{n} \ | \  x>c \} \cup \{ c \}$ 
$\xi_{n}$ - выборка из $p_{n}$. Пусть $L(p_{n})\to 0, \ n\to \infty$
$\eta_{n}=\{ \xi \in\xi_{n} \ | \ \xi \leq c \} \ \ \ \ \zeta_{n}=\{ \xi \in\xi_{n} \ | \ \xi > c \}$
$\xi_{n} = \eta_{n}\cup\zeta_{n}$
$\sigma(f,p_{n}, \xi_{n})=\sigma(f,q_{n},\eta_{n})+\sigma(f,r_{n},\zeta_{n})\implies \int_{a}^{b}fdx=\int_{a}^{c}fdx+\int_{c}^{b}fdx$
  

**Ориентация**: $$\int_{a}^{b}f(x)dx=-\int_{b}^{a}f(x)dx$$
