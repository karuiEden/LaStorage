|**Лемма о вложенных отрезках(признак полноты по Кантору)**:
	$A_{n}$ - вложенные отрезки $\implies \bigcap A_{n} \neq \varnothing,$ т.е. $\exists c\in \mathbb{R}, \ \ \forall n: \ c\in A_{n}$
*Док-во:* сводится к лемме об отделимости
	$\{ A_{n} \}$ - система вложенных отрезков
	$\forall n,m\in \mathbb{N}, n\geq m \ \ [a_{n},b_{n}]\subset[a_{m},b_{m}]$
	$a_{m}\leq a_{n}< b_{n}\leq b_{m} \quad \forall m,n \ \ a_{m}<b_{m}$
	$A:=\{ a_{n} \}, B:=\{ b_{n} \}, \forall n\in \mathbb{N}, \ \ A<B\implies \exists c\in \mathbb{R} \quad\forall m,n \ a_{n}\leq c\leq b_{n}$, в частности при $m=n \ \ \ a_{n}\leq c\leq b_{n} \Leftrightarrow \exists c\in[a_{n},b_{n}]$

|Вложенная система **стягивается**, если длина $[a_{n}, b_{n}]=b_{n}-a_{n}\to 0$ при $n\to \infty$

|Следствие: $\{ A_{n} \} = \{ [a_{n},b_{n}] \}$ стягивается $\implies \exists!c\in \bigcap_{n\in \mathbb{N}}A_{n}\Leftrightarrow \bigcap_{n\in \mathbb{N}}A_{n}=\{ c \}$
*Док-во:* по Кантору $\exists c\in \bigcap A_{n}$
	если бы $\exists c'\neq c, c'\in \bigcap A_{n},$ то $\forall n \ c',c\in[a_{n},b_{n}](\text{для определённости} \ c' < c)\implies$
	$0<c - c'\leq b_{n}-a_{n}\implies$ следовательно длина $[a_{n},b_{n}] \nrightarrow_{n\to \infty} 0$

|К$\implies$Д, т.е. $\forall \{ [a_{n},b_{n}] \}$ вложенных отрезков имеет общую точку $\implies \{ A,B \}$ сечение $\mathbb{R}$ имеет рубеж, т.е. $\exists c\in \mathbb{R}:\forall a\in A,\forall b\in B \ \ \ a\leq c\leq b$
*Док-во:* $\forall a\in A, \forall b\in B$
