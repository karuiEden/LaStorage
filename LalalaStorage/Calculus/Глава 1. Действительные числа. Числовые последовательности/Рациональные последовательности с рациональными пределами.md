|**Определение последовательности:**

$$\forall n \in \mathbb{N} \rightarrow a_{n}(\in \mathbb{Q})\space \{ a_{n} \} \quad a_{n}, n \in \mathbb{N}$$
$\huge{a \in \mathbb{Q}}$ является пределом $\{ a_{n} \}$, если $\large \forall \varepsilon > 0 \space \exists n_{\varepsilon}: \forall n \geq n_{\varepsilon}, |a_{n}-a| < \varepsilon$  
$$\\lim_{ n \to \infty } {a_{n}} = a \quad (n \rightarrow +\infty) $$
Пример такой последовательности может быть $\sqrt{ 2 }$ :
$a_{1} = 1.4$
$a_{2} = 1.41$
$a_{3} = 1.414$

$U_{\varepsilon}(a):=(a - \varepsilon; a + \varepsilon)\quad -$ $\quad \huge{\varepsilon}$-окрестность $a$
$U(a) -$ произвольный открытый промежуток, содержащий $a$

|**Окрестное определение:** $\large a_{n} \rightarrow a$, если
$$\large\forall U_{\varepsilon}(a) \space \exists n_{\varepsilon}: \forall n \geq n_{\varepsilon}, a_{n} \in U_{\varepsilon}(a)$$
$$\large\forall U(a) \space\exists n_{U_{\varepsilon}}: \forall n \geq n_{U_{\varepsilon}}, a_{n} \in U(a)$$
Добавление или исключение конечного множества элементов из последовательности не влияет на подпоследовательность и величину предела.

|Последовательность $\{ a_{n} \}\in \mathbb{Q}$ **сходится**, если она имеет **предел**, т.е.
$$\forall \varepsilon > 0 \space\exists n_{\varepsilon}: \forall n \geq n_{\varepsilon} \quad |a_{n} - a| < \varepsilon$$
**Отрицание**: $$\forall a \exists \varepsilon > 0: \forall N \exists n \geq N \quad  |a_{n} - a| \geq \varepsilon$$ | Каждая последовательность имеет не более одного предела.
*Доказательство*:  от противного. 
![[Pasted image 20241108183845.png]]
Пусть  $\exists \{ a_{n} \}, a_{n} \to a', a_{n} \to a''$
$$a' \neq a'' \quad \varepsilon := \frac{|a' - a''|}{4} > 0$$
$$\exists N': \forall n \geq N' \space |a_{n} - a| < \varepsilon \quad \exists N'': \forall n \geq N'' \space |a_{n} - a| < \varepsilon$$
$$\forall n > max\{N', N''\}$$
$$a_{n} \in U_{\varepsilon}(a') \cap U_{\varepsilon}(a'') = \varnothing$$
		Противоречие.

Последовательность $\{ a_{n} \} \ \in \mathbb{Q}$:
1. **ограничена**, если $\exists! M \quad \forall|a_{n}| < M$
2. **ограничена сверху**, если $\exists M: \forall n\space a_{n} < M$
3. **ограничена снизу**, если $\exists M: \forall n \space a_{n} > M$
| Последовательность ограничена $\Leftrightarrow$ последовательность ограничена и сверху, и снизу

*Док-во:* 
	а) $\{ a_{n} \}$ ограничена $\to$ ограничена сверху и снизу:
	$$\exists M: \forall n \space|a_{n}| < M \Leftrightarrow -M<a_{n}<M$$
	б) $\{ a_{n} \}$ ограничена сверху и снизу $\to$ ограничена:
	$$\exists M, m: \forall n \space m < a_{n} < M |a_{n}| < max\{ |m|, |M| \}$$

|Если последовательность сходится, то она $\{ a_{n} \}$ ограничена.

*Док-во:* $\{ a_{n} \}$

	$a_{n} \to a \space(n \to \infty), \varepsilon := 1$
	$$\exists N_{1}:  \forall n \geq N_{1} \quad |a_{n} - a| < 1 \Leftrightarrow -1 < a_{n} - a < 1 \Leftrightarrow a - 1 < a_{n} < a + 1$$
	$$\forall n: a_{n} < max\{ a_{1}, a_{2}, a_{3}, \dots, a_{N_{1} - 1}, \dots, a + 1 \} := M$$

\end{array}\} \implies \forall n \geq max\{ N_{1}, N_{2} \} |\alpha_{n} + \beta_{n}| < |\alpha_{n}| + |\beta_{n}| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon$$

|**Арифметические операции** над сходящимися последовательностями.
	$\{ a_{n} \}$ и $\{ b_{n} \}$ сходящиеся.
1. $\lim_{ n \to \infty } (a_{n} \pm b_{n}) = \lim_{ n \to \infty } a_{n} \pm \lim_{ n \to \infty } b_{n}$
2. $\lim_{ n \to \infty } (a_{n}b_{n}) = \lim_{ n \to \infty }a_{n} \cdot \lim_{ n \to \infty }b_{n}$
3. если $\lim_{ n \to \infty } b_{n} \neq 0$, то $\lim_{ n \to \infty } \frac{a_{n}}{b_{n}} = \frac{\lim_{ n \to \infty }a_{n}}{\lim_{ n \to \infty }b_{n}}$
	Утверждения: $i)$ пределы слева $\exists$.
				$ii)$ равны тому, что справа.
*Док-во:* предположим $\lim_{ n \to \infty } a_{n} = a \quad \lim_{ n \to \infty }b_{n} = b$ 
					$a_{n} = a + \alpha_{n} \quad b_{n} = b + \beta_{n} \quad (\alpha_{n}, \beta_{n})$ - б.м.
1. $a_{n} \pm b_{n} = (a + \alpha_{n}) \pm (b + \beta_{n}) = (a \pm b) + (\alpha_{n} \pm \beta_{n}) \implies_{n\to \infty} a\pm b$
2. $a_{n}b_{n} = (a + \alpha_{n})(b + \beta_{n})= ab + a\beta_{n} + \alpha_{n}b + \alpha_{n}\beta_{n} \implies_{n\to \infty} \quad ab$ 
3. Если $b \neq 0$, то $\varepsilon := \frac{|b|}{2}$
		$b_{n} = 0$ может быть только для точечного числа членов.
		Считаем, что такие $b_{n}$ отброшены: $\large \frac{a_{n}}{b_{n}} - \frac{a}{b} =\frac{a_{n}b - ab_{n}}{b_{n}b} =\frac{(a + \alpha_{n})b - a(b + \beta_{n})}{b b_{n}} = \frac{1}{b_{n}} \cdot \frac{(a + \alpha_{n})b - a(b + \beta_{n})}{b}$ - вторая дробь б.м.
![[Pasted image 20241111152336.png]]
	Теперь проверим $b_{n}$ на ограниченность:
	$\varepsilon := \frac{|b|}{2} \ \exists n_{\varepsilon} \forall n\geq n_{\varepsilon} \ \ |b_{n} - b| < \frac{|b|}{2} \Leftrightarrow -\frac{|b|}{2} < b_{n} - b < \frac{|b|}{2}$
	1. $b > 0: \quad b_{n} >b - \frac{|b|}{2}=b-\frac{b}{2}=\frac{b}{2}(>0) \implies|b_{n}| > \frac{b}{2}$
	2. $b < 0: \quad b_{n} < b + \frac{|b|}{2}=b-\frac{b}{2}=\frac{b}{2}(<0) \implies |b_{n}| > \frac{b}{2}$
	![[Pasted image 20241111152401.png]]
	$\large \frac{1}{|b_{n}|} < \frac{2}{b}$ , т.е. $\large \left\{  \frac{1}{b_{n}}  \right\}$ **ограничена** $\implies$ 
	$\Large \frac{1}{b_{n}} \cdot \frac{1}{b}(a_{n}b- b_{n}a)$ б.м $\Large \implies \frac{a_{n}}{b_{b}} \to_{n \to \infty} \frac{a}{b}$
