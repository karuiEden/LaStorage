Последовательность $\mathbb{R}$ чисел с $\mathbb{R}$ пределами **аналогично** $\mathbb{Q}$:
Определения:
1. Последовательность
2. Предел последовательности, бесконечно малая
3. Сходимость последовательностей
4. Ограниченность последовательностей
Теоремы:
1. Единственность предела
2. Ограниченность сходящийся последовательнстей
3. $a_{n}\longrightarrow a, \ \ a_{n}=a + \alpha_{n}$
4. Арифметические свойства последовательностей


|(О пределе стационарной последовательности) Если $\{ a_{n} \} \ \exists N:$
$\forall n\geq N\quad a_{n}=a$, то $\{ a_{n} \}$ сходится и $a_{n} \longrightarrow a$
*Док-во:* $\forall\varepsilon>0 \ \  n_{\varepsilon}:=N,\ \forall n\geq n_{\varepsilon} \ \ |a_{n}-a|=0<\varepsilon$

|Теорема о предельных переходах:
	 $a_{n}\longrightarrow a, \ b_{n}\longrightarrow b$, если $\forall n\geq N \ \ \ a_{n}\leq b_{n}$, то $a\leq b$
*Док-во:* Пусть $a>b$
![[Pasted image 20241112204634.png]]
$\varepsilon = \frac{a-b}{2}: \exists N_{1}\in \mathbb{R}, \ \forall n\geq N_{1} \ |a_{n}-a|<\varepsilon \Leftrightarrow -\varepsilon < a_{n}-a<\varepsilon \Leftrightarrow a-\varepsilon<a_{n}<a+\varepsilon$
$\exists N_{2}\in \mathbb{R}, \forall n\geq N_{2}: |b_{n}-b|<\varepsilon\Leftrightarrow -\varepsilon<b_{n}-b<\varepsilon \Leftrightarrow b-\varepsilon<b_{n}<b+\varepsilon$
$\forall \max\{ N_{1},N_{2} \}: \ b_{n}<b+\varepsilon=a-\varepsilon<a_{n}\implies b_{n}<a_{n}\implies$ противоречие.

Замечание: Строгое неравенство может перейти в нестрогое

Замечание: Если $a_{n}\longrightarrow a, \forall n \quad a_{n}\geq b\implies a\geq b,a_{n}>b\implies a \geq b$

| Пусть $a_{n}\longrightarrow a, \ b_{n}\longrightarrow b$ и $a<b \implies \exists N: \forall n\geq N \ \ \  a_{n}<b_{n}$
*Док-во*: $\varepsilon:=\frac{b-a}{2}$
![[Pasted image 20241112210856.png]]
$\exists N_{1}: \forall n\geq N_{1}  \ \ |a_{n}-a|<\varepsilon \Leftrightarrow a_{n}<a+\varepsilon$
$\exists N_{2}: \forall n\geq N_{2} \ \ |b_{n}-b|<\varepsilon \Leftrightarrow b-\varepsilon < b_{n}$
$\forall n\geq max\{ N_{1},N_{2} \} \ \ a_{n}<a+\varepsilon=b-\varepsilon < b_{n}\implies a_{n}<b_{n}$

|Следствие(теорема о сохранении знака): если $a_{n}\longrightarrow a\neq_{0}$, то, начиная с некоторого номера $sign \ a_{n} = sign \ a$
$$\triangleq sign \ a = sgn \ a:=\begin{cases} 
1: a> 0, \\
0:a=0, \\
-1:a<0
\end{cases}$$
*Док-во:*
$\varepsilon:=\frac{|a|}{2}, \exists N: \forall n\geq N \ \ |a_{n}-a|<\varepsilon \Leftrightarrow a-\varepsilon<a_{n}<\varepsilon+a\Leftrightarrow a-\frac{|a|}{2}<a_{n}<a+\frac{|a|}{2}$ 
пусть $a>0, |a|=a: \frac{a}{2}<a_{n}< \frac{3a}{2}\implies a_{n}>0$
пусть $a<0, |a|=-a: a_{n}<a- \frac{a}{2}=\frac{a}{2}\implies a_{n}<0$

|Теорема о зажатой последовательности: $\{ a_{n} \}, \{ b_{n} \}, \{ c_{n} \}$
	$\forall n \ a_{n}\leq b_{n}\leq c_{n}$ , если $a_{n}\longrightarrow a, c_{n}\longrightarrow a$, тогда $\{ b_{n} \}$ сходится и стремится к $a$ 
*Док-во*: 
$\forall\varepsilon,\varepsilon:=const, \exists N_{1}:\forall n\geq N_{1} \ \ |a_{n}-a|<\varepsilon\Leftrightarrow a-\varepsilon<a_{n}$
		$\exists N_{2}:\forall n\geq N \ \ |c_{n}-a|<\varepsilon \Leftrightarrow c_{n}<a+\varepsilon$
$\exists N=max\{ N_{1},N_{2} \}: \forall n\geq N, \ \ a-\varepsilon<a_{n}\leq b_{n}\leq c_{n}<a+\varepsilon\implies a-\varepsilon<b_{n}<a+\varepsilon\implies|b_{n}-b|<\varepsilon$
|Теорема о сходимости модуля: Пусть $a_{n}\longrightarrow a\implies \{ |a_{n}| \}$ сходится к $|a_{n}|\longrightarrow|a|$
	**Неравенство треугольников: $|a+b| \leq |a|+|b|,\ \ \ a,b\in \mathbb{C}$ 
	![[Pasted image 20241115122702.png]]
1. $a,b\in \mathbb{R}$
	$a\leq|a|, b\leq|b|\implies a+b\leq|a|+|b|$
	$-a\leq|a|,-b\leq|b|\implies-(a+b)\leq|a|+|b|\Leftrightarrow a+b\geq-(|a|+|b|)$ 
	Следовательно, $|a+b|\leq|a|+|b|$
2. $|a-b|\geq||a|-|b||$
	$|a|=|a+b-b|\leq|a-b|+|b|\implies|a|-|b|\leq|a-b|$
	$|b|=|b-a+a|\leq|b-a|+|a|\implies|b|-|a|\leq|a-b|\Leftrightarrow|a|-|b| \geq-|a-b|$
	$-|b-a|\leq|a|-|b|\leq|b-a|\Leftrightarrow||a|-|b||\leq|a-b|$
*Док-во:*$\forall\varepsilon>0, \exists n_{\varepsilon}>0:\forall n\geq n_{\varepsilon} \ \ |a_{n}-a|<\varepsilon,||a_{n}|-|a||\leq|a_{n}-a|\implies||a_{n}|-|a||<\varepsilon$

|$\{ a_{n} \}$ - бесконечно большая последовательность, если $\forall \varepsilon>0,\exists n_{\varepsilon}>0:\forall n\geq n_{\varepsilon} \ \ |a_{n}|>\varepsilon$
$a_{n}\longrightarrow\infty,\lim_{ n \to \infty }a_{n}=\infty \ \ \{ a_{n} \}$ - б.б

|$a_{n}\longrightarrow+\infty,$ если $\forall\varepsilon>0,\exists n_{\varepsilon}>0:\forall n\geq n_{\varepsilon} \ \ a_{n}>\varepsilon$
 $a_{n}\longrightarrow-\infty$, если $\forall\varepsilon>0,\exists n_{\varepsilon}>0:\forall n\geq n_{\varepsilon} \ \ a_{n}<-\varepsilon$
 $a_{n}=(-a)^nn\longrightarrow\infty$, не к $+\infty$, не к $-\infty$

|Теорема о связи б.б. и б.м. последовательностях: 
1. $\{ \alpha_{n} \}$ - б.м. и $\alpha_{n}\neq 0, \ \forall n: \left\{  \frac{1}{\alpha_{n}}  \right\}$-б.б
2. $\{ a_{n} \}$ - б.б и $a_{n}\neq 0, \ \forall n:\left\{  \frac{1}{a_{n}}  \right\}$-б.м.
*Док-во:*
1. $\forall\varepsilon>0, \ \exists n_{\varepsilon}>0:\forall n\geq n_{\varepsilon} \ \ |\alpha_{n}|< \frac{1}{\varepsilon}\implies |\frac{1}{\alpha_{n}}|>\varepsilon$
2. $\forall\varepsilon>0, \exists n_{\varepsilon}>0:\forall n\geq n_{\varepsilon} \ |a_{n}|>\varepsilon\implies | \frac{1}{a_{n}}|<\varepsilon$
Примеры:
1. $n^p,p>0$ б.б
	$\forall \varepsilon>0, n^{p}>\varepsilon\Leftrightarrow n>\varepsilon^{\frac{1}{p}}$ определено по принципу Архимеда
2. $a^n,\  a>1$ б.б
	$a^{n}=(1+q)^{n}=1+nq+ \frac{n(n-1)}{2}q^{2}+\dots+q^{n}>qn, \ \ n\geq 2$ б.б.
$n^{p},\ p>0$ и $a^{n} \ a>0$ б.б. $\Leftrightarrow \{n^{-p}\},\ \{ a^{-n} \}$ б.м.
