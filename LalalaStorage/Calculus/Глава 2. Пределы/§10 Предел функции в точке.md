| Теорема: $f(x)$ имеет предел по Коши $\Leftrightarrow$ $f(x)$ имеет предел по Гейне.
*Док-во:* К $\Rightarrow$ Г;  $f(x) \longrightarrow A, \ x \longrightarrow a$:
	$\forall \{ x_{n} \} \quad x_{n} \longrightarrow a$ 
	$\forall\varepsilon>0 \ \exists \delta_{\varepsilon}: \forall x \quad 0<|x-a|<\delta_{\varepsilon} \ \ \ |f(x)-A|<\varepsilon$ 
	$\exists N_{\delta_{\varepsilon}}(=N_{\varepsilon}): \forall n\geq N_{\varepsilon} \quad |x_{n}-a|<\delta_{\varepsilon}\implies|f(x_{n}) - A|<\varepsilon$ 
Г$\Rightarrow$ К: 
1. Сначала проверим, что $\forall \{ x_{n} \}$ предел $\{ f(x_{n}) \}$ один и тот же.
$\exists \{ x_{n}' \}, \{ x_{n}'' \}$ - последовательность Гейне
$f(x_{n}')\longrightarrow A', \ \ f(x_{n}'')\longrightarrow A''$
Перетасовываем $\{ x_{n}' \}$ и $\{ x''_{n} \}$ 
$$x_{n}=\begin{cases}
x_{k}', \ n=2k-1 \\
x''_{k}, \ n=2k
\end{cases}\ , n, \ k \in \mathbb{N}$$
$x_{n}\longrightarrow a$ и последовательность Гейне
	1)$x_{n}\longrightarrow a \ \forall\varepsilon>0 \ \ \exists N',N''$, $N=2(\max\{ N',N'' \})$
	$\forall n>N' \ \ |x_{n}'-a|<\varepsilon$
	$\forall n > N'' \ \ |x_{n}''-a|<\varepsilon$
Проверим, что $\forall n>N \ \ |x_{n}-a|<\varepsilon$
		i) $n=2k \implies n>2(\max\{ N',N'' \} + 1), \ \ k>\max\{ N',N'' \}+1, k>N''\implies|x_{n}-a|=|x_{k}''-a|<\varepsilon$
		ii)$n=2k-1\implies n>2(\max\{ N',N'' \}+1),2k-1>2(\max\{ N',N'' \}+1), 2k-1>N'\implies |x_{n}-a|=|x_{k}'-a|<\varepsilon$
	2)$x_{n}\longrightarrow a$
	$\forall n \ \ x_{n}=x'_{k}\neq a, \ \ \ x''_{k}\neq a$
	$\forall n  \ x_{n}\neq a\implies \ \ \{ x_{n} \}$ последовательность Гейне
	Следовательно $\{ f(x_{n}) \}$ сходится, тогда $f(x'_{k})\longrightarrow A', \ \ f(x''_{k})\longrightarrow A'' -$ противоречие. Значит, для любой последовательности.
	Предел обозначим как $A$ 
Проверим $f(x)\longrightarrow A (x\longrightarrow a)$ в смысле Коши:
	$\exists \varepsilon>0 \ \forall\delta_{\varepsilon}>0 \ \exists x: \ 0<|x-a|<\delta, \ \ |f(x)-A|\geq\varepsilon$
		$\delta_{1}:=\frac{1}{2},  \ \exists x_{1}: \ 0<|x_{1}-a|<\delta_{1}, \ |f(x_{1})-A|\geq\varepsilon$
		$\delta_{2}=\frac{1}{2^{2}}, \ \exists x_{2}: 0<|x_{2}-a|<\delta_{2}. \ |f(x_{2})-A|\geq\varepsilon$
		$\dots{}$
		$\delta_{n}:=\frac{1}{2^{n}}, \ \exists x_{n}: 0<|x_{n}-a|<\delta_{n}, \ |f(x_{n})-A|\geq\varepsilon$
$\{ x_{n} \}$ последовательность Гейне: $\forall n \ x_{n}\neq a, \ \ x_{n}\longrightarrow a, \ \ n\longrightarrow\infty\implies f(x_{n})\longrightarrow A$
$\forall n \ \ |f(x_{n})-A|\geq\varepsilon(>0)\implies$ противоречие $\implies$ предположение, что $A$ не предел по Коши приводит к противоречию.
