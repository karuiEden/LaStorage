|$f(x)$ в $\dot{U}(a) \ \exists$ конечный предел $\lim_{x\longrightarrow a} f(x) \Leftrightarrow \ \forall\varepsilon>0, \ \exists\delta_{n}>0: \ \ \forall x',x''\in \dot{U}_{\varepsilon}(a) \ \ |f(x')-f(x'')|<\varepsilon$ - условие Коши.

*Док-во:* (предлагается существование $\exists \lim_{ x \to a }f(x)$)
$\Rightarrow)$ Проверка с использованием неравенства треугольников.
$A:\lim_{ x \to a }f(x) \ \forall\varepsilon>0, \ \exists\delta_{\varepsilon}>0: \forall x \ \ 0<|x-a|<\delta_{\varepsilon}'  \ \ |f(x)-A|< \frac{\varepsilon}{2}$
$\forall x',x''\in \dot{U}_{\delta_{\varepsilon}}(a): \ |f(x')-f(x'')|=|((f(x')-A)-(f(x'')-A)|\trianglelefteq|f(x')-A|+|f(x'')-A|< \frac{\varepsilon}{2} + \frac{\varepsilon}{2}=\varepsilon$
$\Leftarrow)$ Из выполнения условия Коши следует существование предела $f(x)$ при $x\longrightarrow a$ в смысле Гейне.
Сводится к критерию Коши существования предела последовательности, проверка $\forall \{ x_{n} \} \ \ x_{n}\longrightarrow a$ последовательность Гейне $\{ f(x) \}$ фундаментальна. $\forall \{ x_{n} \}$ последовательности Гейне: $\forall\varepsilon>0, \ \exists\delta_{\varepsilon}>0 \ \ \forall x',x'' \in \dot{U}_{\delta}(a) \ \ |f(x')-f(x'')|<\varepsilon$
$\exists n_{\delta_{\varepsilon}}(=n_{\varepsilon})\ \ \forall n\geq n_{\varepsilon} \ \ |x_{n}-a|<\delta_{\varepsilon} \ \ x_{n}\neq a\implies \{ x_{n} \}$ - последовательность Гейне.
$\forall m,n\geq n_{\varepsilon}\implies x_{n},x_{m}\in \dot{U}_{\delta}(a)\implies|f(x_{n})-f(x_{m})|<\varepsilon$ 
Значит, последовательность Гейне фундаментальна $\implies$ сходится $\implies$ выполнено определение предела по Гейне.

|Отрицание  выполнения признака Коши:
$\exists a>0, \ \forall\varepsilon>0 \ \ \exists x',x''\in \dot{U_{\delta}}(a) \ |f(x')-f(x'')|\geq\varepsilon$
