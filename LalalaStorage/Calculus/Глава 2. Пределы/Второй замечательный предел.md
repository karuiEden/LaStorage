$$\lim_{ x \to \infty }\left( 1+ \frac{1}{x} \right)^{x}=e \quad (x\longrightarrow+\infty, \ \ \ x\longrightarrow-\infty) $$
*Док-во:* $e:=\lim_{ x \to \infty }\left( 1+ \frac{1}{x} \right)^{x}$
1. $x\longrightarrow+\infty$
	$\left( 1+ \frac{1}{n+1} \right)^{n}=\frac{\left( 1+ \frac{1}{n+1} \right)^{n+1}}{1+ \frac{1}{n+1}}\longrightarrow \frac{e}{1}=e$
	$\left( 1+ \frac{1}{n} \right)^{n+1}=\left( 1+ \frac{1}{n} \right)^{n}\times\left( 1+ \frac{1}{n} \right)\longrightarrow e\times 1=e$
	$x\geq 1: [x]\leq x<[x]+1$
	$\frac{1}{[x]+1}< \frac{1}{x}< \frac{1}{[x]} \ \ \ \ 1+ \frac{1}{[x]+1}< \frac{1}{x}+1< 1+ \frac{1}{[x]}\quad [x]\in \mathbb{N}$
	$\left( 1+ \frac{1}{[x]} \right)^{[x]+1}>\left( 1+ \frac{1}{x} \right)^{x}>\left( 1+ \frac{1}{[x]+1} \right)^{[x]}$
	$e+\varepsilon>\left( 1 + \frac{1}{n} \right)^{n+1}\geq\left( 1+ \frac{1}{n+1} \right)^{n}>e-\varepsilon$
	$\forall\varepsilon>0 \ \ \exists N_{\varepsilon}: \forall n\geq N_{\varepsilon} \ \ \ \forall x\geq N_{\varepsilon}+1 \ \ \ [x]\geq N_{\varepsilon}$
	$e+\varepsilon>\left( 1+ \frac{1}{[x]} \right)^{[x]+1}>\left( 1+ \frac{1}{x} \right)^{x}>\left( 1+ \frac{1}{[x]+1} \right)^{[x]}>e-\varepsilon\implies$
	$\left( 1+ \frac{1}{x} \right)^{x}\longrightarrow\varepsilon \ \ \ \ x\longrightarrow+\infty$ по определению
2. $x\longrightarrow-\infty$
	Замена $t:=-x$
	$\lim_{ x \to -\infty }\left( 1 + \frac{1}{x} \right)^{x}=\lim_{ t \to +\infty }\left( 1- \frac{1}{t} \right)^{-t}=\lim_{ t \to +\infty }\left( \frac{t}{t-1} \right)=\lim_{ t \to +\infty }\left( \frac{(t-1)+1}{(t-1)} \right)^t=\lim_{ t \to +\infty }\left( 1+ \frac{1}{t-1} \right)^{(t-1)+1}$
	$\lim_{ t \to +\infty }\left( 1+ \frac{1}{t-1} \right)^{t-1}\left( 1+ \frac{1}{t-1} \right)=e$ 
	$\lim_{ x \to +\infty }\left( 1 + \frac{1}{x} \right)^{x}=\lim_{ x \to -\infty }\left( 1 + \frac{1}{x} \right)^{x}=e\Leftrightarrow\lim_{ x \to \infty }\left( 1 + \frac{1}{x} \right)^{x}=e$ 
Следствие: $\lim_{ x \to 0 }(1+x)^{\frac{1}{x}}=e$
