|$f(x)$ на $<a,b>\subset \mathbb{R}, \ \ \ f(x)$ **монотонно возрастает** на $<a,b>$, если $\forall x_{1},x_{2}\in<a,b> \ \ x_{1}<x_{2}\implies f(x_{1})\leq f(x_{2})$ и **строго монотонно возрастает** на $<a,b>$, если $\forall x_{1},x_{2}\in<a,b> \ \ x_{1}<x_{2}\implies f(x_{1})<f(x_{2})$

|$f(x)$ на $<a,b>\subset \mathbb{R}, \ \ f(x)$ **монотонно убывает** на $<a,b>,$ если $\forall x_{1},x_{2}\in<a,b> \ \ x_{1}>x_{2}\implies f(x_{1})\geq f(x_{2})$ и **строго монотонно убывает** на $<a,b>$, если $\forall x_{1},x_{2}\in<a,b> \ \ \ x_{1}>x_{2}\implies f(x_{1})>f(x_{2})$

|$f(x)\nearrow$ на $(a,b)$. Тогда $\exists \lim_{ x \to b-0 }f(x)$ и $\lim_{ x \to a+0 }f(x)$ (конечные/бесконечные)
	$\lim_{ x \to b-0 }f(x)=\sup_{(a,b)}f(x) \quad \lim_{ x \to a+0 }f(x)=\inf_{(a,b)}f(x)$
*Док-во:* по определению пределов
	$A:=\sup_{(a,b)} f(x), \ \ A\in \mathbb{R}, \ A=_{\infty}$
1. $A\in \mathbb{R}$ по определению $\sup$ $\forall\varepsilon>0 \ \ \exists x_{\varepsilon} \ \ A-\varepsilon<f(x_{\varepsilon})<A$ по определению монотонного возрастания $\forall x \in[x_{\varepsilon},b) \ \ \ A-\varepsilon<f(x_{\varepsilon})\leq f(x)\leq A, \ \ \delta=b-\varepsilon; \ \ \forall x \ \ b-\delta<x<b\implies \ \ A-\varepsilon<f(x)\leq A\implies|f(x)-A|<\varepsilon$, т.е. по определению $\lim_{ x \to b-0 }f(x)=A$
2. Пусть $\sup_{(a,b)}f(x)=A=+\infty$ По определению $\forall\varepsilon>0 \ \ \exists x_{\varepsilon}\in(a,b): \ \ \ f(x_{\varepsilon})>\varepsilon\implies \forall x \in[x_{\varepsilon},b)\implies f(x)\geq f(x_{\varepsilon})>\varepsilon$, т.е по определению $f(x)\longrightarrow+\infty$ при $x\longrightarrow b-0$
|Следствие: если $f(x)\searrow\ \ (a,b)$, то $\exists \lim_{ x \to b-0 }f(x)$, $\lim_{ x \to a+0 }f(x)$, то $\lim_{ x \to b-0 }f(x)=\inf_{(a,b)}f(x), \ \ \lim_{ x \to a+0 }f(x)=\sup_{(a,b)}f(x)$

|Следствие: если $f(x)$ монотонна на $(a,b) \ \ \ \forall c\in(a,b) \ \ \ f(x)$ имеет конечные пределы при $x\longrightarrow c+0$ и $x\longrightarrow c-0$
*Док-во:* $f(x)$ монотонна на $(a,c)$ и $(c,b)$, и $\forall x'\in(a,c)$ и $\forall x''\in(c,b)$.
если $f(x)\nearrow\quad f(x')\leq f(c)\leq f(x'')$, $f(x)\searrow\quad \ \ f(x')\geq f(c)\geq f(x'')$
