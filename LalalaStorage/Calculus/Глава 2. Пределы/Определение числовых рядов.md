$\forall \{ a_{n} \} \mathbb{R}$ или $\mathbb{C}$ $p\leq q\quad \ p,q\in \mathbb{N}$
$$\sum_{n=p}^{q}a_{n} = a_{p}+\dots+a_{q}$$
$a_{1}+a_{2}+a_{3}+\dots+a_{n}+\dots=\sum_{n=1}^{\infty}a_{n}\left( \sum_{n=0}^{\infty} \right)$

$$|S:=\sum_{k=1}^{n}a_{k} - \text{частичная сумма ряда}$$
|Если $\{ S_{n} \}$ **сходится**, то $\sum_{1}^{\infty}a_{k}$ сходится и $\sum_{1}^{\infty}a_{k}:=\lim_{ n \to \infty }S_{n}$
| Если предела нет, то ряд **расходится**
$$\{ a_{n} \}, \ a_{n}:=\sum_{k=2}^{\infty}(a_{k}-a_{k-1})+a_{1};\ \{ a_{n} \} \text{ - частичная сумма } a_{1}+\sum_{2}^\infty(a_{k}-a_{k-1})$$
|Критерий Коши сходимости ряда: $\sum_{1}^{\infty}a_{k}$ сходится $\Leftrightarrow \forall\varepsilon>0 \ \exists n_{\varepsilon}: \forall n\geq n_{\varepsilon}, \ \forall p \in \mathbb{N}_{0}\implies|\sum_{k=n}^{n+p}a_{k}|<\varepsilon$
*Док-во:* $\sum a_{k}$ сходится по определению $\implies$
$\exists \lim_{ n \to \infty }\sum_{1}^{n}a_{k}=\lim_{ n \to \infty }S_{n}\Leftrightarrow\forall\varepsilon>0, \ \exists n_{\varepsilon}: \forall n\geq n_{\varepsilon}, \ \forall p \in \mathbb{N}_{0}, |S_{n}-S_{n+p}|<\varepsilon$
$|\sum_{k=1}^na_{k}-\sum_{k=1}^{n+p}a_{k}|=|\sum_{k=n+1}^{n+p}a_{k}|<\varepsilon$

|Следствие 1: Изменение конечного числа членов ряда не влияет на сходимость.
*Док-во:* $\exists N,$ что $a_{n}$ с $n\geq N$ не меняется, если брать $n_{\varepsilon}\geq N$, то $\sum_{n}^{n+p}a_{n}$ не изменится.

|Следствие 2 (необходимое условие сходимости): Если $\sum a_{k}$ сходится $\implies a_{k}\longrightarrow 0, \ k\longrightarrow\infty$
*Док-во:*
1. Из критерия Коши. Возьмём $p=0, \ |\sum_{n}^{n+p}a_{k}|=|a_{n}|<\varepsilon$
2. $\lim_{ n \to \infty }a_{n}=\lim_{ n \to \infty }(S_{n}-S_{n-1})=\lim_{ n \to \infty }S_{n}-\lim_{ n \to \infty }S_{n-1}=0$

Примеры: 
1. $1+q^1+q^{2}+q^{3}+\dots+q^n+\dots$ - геометрическая прогрессия
	1.  Если $|q|\geq 1\implies \forall n \ |q^{n}|\geq 1\implies$ ряд расходится
	2. Если $|q|<1\implies \forall n \ |q^n|<1\implies$ ряд сходится
2. $\sum_{1}^{\infty} \frac{1}{n}$ - гармонический ряд
	$\frac{2}{c}= \frac{1}{a}+ \frac{1}{b}\implies c$ - среднее гармоническое $a$ и $b$
	$c = \frac{2}{ \frac{1}{a}+ \frac{1}{b}} = \frac{2ab}{a+b}; \sum\frac{1}{n^{p}}: \frac{1}{n-1}, \frac{1}{n}, \frac{1}{n+1}$
	$\sum_{1}^\infty$ расходится по критерию Коши
3. $\sum_{1}^\infty(-1)^n$ расходится (необходимое условие)
