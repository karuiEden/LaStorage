$f(x)$ на $(c,d)\setminus \{ a \}$

|Предел $f(x)$ единственный
*Док-во:*
![[Pasted image 20241214155925.png]]
От противного: Пусть $f(x)\longrightarrow A_{1}, \ f(x)\longrightarrow A_{2}, \ \ x\longrightarrow a,\ \ A_{1}\neq A_{2}$
$\varepsilon:=\frac{|A_{1}-A_{2}|}{4}, \ \exists\delta_{1}>0:\forall x \ \ 0<|x-a|<\delta_{1} \ \ |f(x)-A_{1}|<\varepsilon$, аналогично $|f(x)-A_{2}|<\varepsilon$ 
$\delta:=\min\{ \delta_{1},\delta_{2} \} \ \ \forall x \in U_{\delta}(a)\implies$ противоречие

|Если при $x\longrightarrow a,  \ \ f(x)\longrightarrow A$, то $|f(x)|\longrightarrow|A|$ при $|x|\longrightarrow|a|$
*Док-во:* Непосредственно: $\forall\varepsilon>0, \ \ \exists\delta_{\varepsilon}>0:\forall x \ \ 0<|x-a|<\delta_{\varepsilon} \ \ ||f(x)-|A||\leq|f(x)-A|<\varepsilon$

|Теорема сохранения знаков: Если $f(x)\longrightarrow A\neq 0$ при $x\longrightarrow a$. Тогда $\exists \dot{U}(a)$, в которой знак $f(x)$ будет таким же как и у $A$
*Док-во:* $\varepsilon:= \frac{|A|}{2} \ \ \exists\delta_{\varepsilon}>0 \ \ \forall x: \ \ 0<|x-a|<\delta_{\varepsilon} \ \ \ |f(x)-A|<\varepsilon\implies A- \frac{|A|}{2}<f(x)<A+ \frac{|A|}{2}$
1. $A>0:f(x)>A- \frac{|A|}{2}\Leftrightarrow \ f(x)> \frac{A}{2}>0$
2. $A<0:f(x)<A+ \frac{|A|}{2}\Leftrightarrow f(x)< \frac{A}{2}<0$
|Локальная ограниченность. Если $\exists \lim_{ x \to a }f(x)$, то $f(x)$ ограничена в некоторой $\dot{U}(a)$
*Док-во:* $A:=\lim_{ x \to a }f(x), \ \ \varepsilon:=1, \ \exists\delta>0 \ \forall x \ \ \ 0<|x-a|<\delta \ \ |f(x)-A|<\varepsilon\Leftrightarrow A-\varepsilon<f(x)<A+\varepsilon\Leftrightarrow A-1<f(x)<A+1$ - ограниченность в $\dot{U}(a)$

|Теорема об арифметических свойствах: $\exists \lim_{ x \to a }f(x), \ \ \ \exists \lim_{ x \to a }g(x)$. Тогда:
1. $\lim_{ x \to a }(f(x)+g(x))=\lim_{ x \to a }f(x)+\lim_{ x \to a }g(x)$
2. $\lim_{ x \to a }(f(x)g(x))=\lim_{ x \to a }f(x)\cdot \lim_{ x \to a }g(x)$
3. Если $\lim_{ x \to a }g(x)\neq 0$ и $g(x)\neq 0$ на области определения, то $\lim_{ x \to a } \frac{f(x)}{g(x)}= \frac{\lim_{ x \to a }f(x)}{\lim_{ x \to a }g(x)}$
*Док-во:* Эквивалентность по Гейне и по Коши.

|Предельный переход в неравенствах: Пусть $f(x)\leq g(x), \ \ x \in(c,d)\setminus \{ a \}$. Если $\exists \lim_{ x \to a }f(x),  \ \ \exists\lim_{ x \to a }g(x)\implies \ \lim_{ x \to a }f(x) \leq \lim_{ x \to a }g(x)$ 
*Док-во:* Пусть $f(x)\longrightarrow A, \ g(x)\longrightarrow B$ при $x\longrightarrow a$
![[Pasted image 20241214162503.png]]
От противного: Предположим, что $B>A: \ \ \varepsilon:= \frac{A-B}{2}$
$\exists\delta_{1}>0 \ \forall x \ \ 0<|x-a|<\delta_{1} \ \ |f(x)-A|<\varepsilon\Leftrightarrow A-\varepsilon<f(x)<A+\varepsilon$
$\exists\delta_{2}>0 \ \ \forall x \ \ 0<|x-a|<\delta_{2} \ \ |g(x)-B|<\varepsilon\Leftrightarrow B-\varepsilon<g(x)<B+\varepsilon$
$\delta:=\min\{ \delta_{1}, \ \delta_{2}\} \ \ 0<|x-a|<\delta$
$f(x)>A-\varepsilon=A- \frac{A-B}{2}= \frac{A+B}{2}=B+\varepsilon>g(x)$

|О зажатой функции:
1. $\forall x \in(c,d)\setminus \{ a \} \ \ f(x)\leq \phi(x)\leq g(x)$
2. $\exists \lim_{ x \to a }f(x), \ \ \exists \lim_{ x \to a }g(x)$ и $\lim_{ x \to a }f(x)=\lim_{ x \to a }g(x)$, тогда $\exists \lim_{ x \to a }\phi(x), \ \ \lim_{ x \to a }\phi(x)=\lim_{ x \to a }f(x)=\lim_{ x \to a }g(x)$
*Док-во:* По Гейне:
Пусть $\{ x_{n} \}$ последовательность Гейне, т.е. $x_{n}\longrightarrow a,  \ \ \forall n \ \ x_{n}\neq a$
$f(x_{n})<g(x_{n}), \ \ \exists\lim_{ x \to a }f(x_{n}), \ \ \exists\lim_{ x \to a }g(x_{n}), \ \ \ \lim_{ x \to a }f(x_{n})=\lim_{ x \to a }g(x_{n})\implies\text{(по т. о зажатой последовательности) }$
$\exists\lim_{ x \to a }\phi(x_{n})$ и $\lim_{ x \to a }f(x_{n})=\lim_{ x \to a }g(x_{n})=\lim_{ x \to a }\phi(x_{n})$
$\forall \{ x_{n} \}$ последовательности   Гейне $\exists \lim_{ x \to a }\phi(x_{n})\implies$
$\text{по теореме о эквивалентности} \ \ \exists \lim_{ x \to a }\phi(x)$ и $\lim_{ x \to a }f(x)=\lim_{ x \to a }\phi(x)=\lim_{ x \to a }g(x)$
