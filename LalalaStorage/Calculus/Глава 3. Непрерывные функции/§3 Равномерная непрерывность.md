|$f(x)$ на $<a,b>\quad x_{0}\in<a,b> \quad f(x)$ непрерывна в $x_{0}$ относительно $<a,b>$, если $\forall\varepsilon>0 \ \exists\delta_{\varepsilon,x_{0}}>0:\forall x:|x-x_{0}|<\delta_{\varepsilon,x_{0}}, \ x\in<a,b> \ \ \ \ |f(x)-f(x_{0})|<\varepsilon$

Примеры: 
1. $f(x)=x \  \ \forall x\in \mathbb{R} \ \delta:=\varepsilon$
2. $f(x)= \frac{1}{x}$ на $(0,1)\quad x_{1},\frac{x}{2}\in(0,1)\quad |f(x)-f\left( \frac{x}{2} \right)|=| \frac{1}{x}- \frac{2}{x}|=\frac{1}{x}\longrightarrow\infty, \ \ x\longrightarrow 0\quad \delta_{\varepsilon,x}$ зависит от $x$

|$f(x)$ на $<a,b>, \ f(x)$ **равномерно непрерывна** на $<a,b>$, если $\forall\varepsilon>0 \ \exists\delta_{\varepsilon}>0, \ \ \forall x',x''\in<a,b>, \ \ |x'-x''|<\delta_{\varepsilon} \ \implies|f(x')-f(x'')|<\varepsilon$

|Теорема Кантора: Если $f(x)\in C[a,b]$, то $f(x)$ равномерно непрерывна на $[a,b]$
*Док-во:* от противного
	Пусть $f(x)\in C[a,b]$, но не равномерно непрерывна, т.е. $\exists\varepsilon>0\ \forall\delta_{\varepsilon}>0 \ \exists x',x''\in<a,b>, \ |x'-x''|<\delta_{\varepsilon}$, то $|f(x')-f(x'')|\geq\varepsilon$ 
	$\forall n,\delta_{n}:= \frac{1}{n} \ \ \exists x'_{n},x_{n}''\in<a,b> \ \ \ |x_{n}'-x_{n}''|< \frac{1}{n} \ \ \ \ |f(x')-f(x'')|\geq\varepsilon$
	$\{ x_{n}' \}, \{ x_{n}'' \}$ точек $[a,b]$ ограничены, по *B-W* $\exists\{ x_{n_{k}}' \}$ сходится
	$x_{n_{k}}' \longrightarrow c\in[a,b], \ |x_{n_{k}}'-x_{n_{k}}''|< \frac{1}{n_{k}}\leq \frac{1}{k}, \ \ \ x_{n_{k}}'\longrightarrow c\implies$
	$x_{n_{k}}'- \frac{1}{k}<x_{n_{k}}''<x_{n_{k}}'+ \frac{1}{k}\implies x_{n_{k}}'' \longrightarrow c$( по теореме о зажатой последовательности)
	В точке  $c\in[a,b] \ \ f(x)$ непрерывна, $x_{n_{k}}' \longrightarrow c, \ x_{n_{k}}''\longrightarrow c$ по определению непрерывности по Гейне $f(x_{n_{k}}')\longrightarrow f(c) \wedge f(x_{n_{k}}'')\longrightarrow c$ противоречит $|f(x_{n_{k}}')-f(x_{n_{k}}'')|\geq\varepsilon>0 \ \ \ \forall k$
