1. $f(x)=o(\phi(x))$ при $x\longrightarrow a$, если $f(x)=\alpha(x)\cdot \phi(x), \alpha(x)$ б\м при $x\longrightarrow a$
2. $f(x)=O(\phi(x))$ при $x\longrightarrow a$, если $\exists c<0 \wedge U(a): |f(x)|\leq c|\phi(x)| \ \ \ \forall x\in \dot{U}(a)$


1. $o$ и $O$ **оценки**, а не функции
2. бессмысленно без $x\longrightarrow a$

Примеры:
1. $x^{2}=o(x) \ \ x\longrightarrow 0$
	$x^{2}=x\cdot x=\alpha(x)\cdot x=o(x)$
2. $x=o(x^{2}) \ \  x\longrightarrow\infty$
	$x = \frac{1}{x}\cdot x^{2}=\alpha(x)\cdot x^{2}=o(x^{2})$
3. $o(\phi(x))=O(\phi(x)) \ \ x\longrightarrow a \quad f(x)=\alpha(x)\cdot \phi(x) \ \ f(x)=o(\phi(x))$
	$\exists U(a):\forall x\in \dot{U}(a) \ \ |\alpha(x)|<1\implies |f(x)|=|\alpha(x)\cdot \phi(x)|\leq 1|\phi(x)|$ 
4. $O(\phi(x))\neq o(\phi(x)) \ \ \ x\longrightarrow a$
	$x=O(x)\neq o(x) \ \ \ x\longrightarrow 0$
5. $o(1) \ \ x\longrightarrow a$ - б/м при $x\longrightarrow a$
	$f(x)=o(1)$ при $x\longrightarrow a, \ \ f(x)=\alpha(x)\cdot 1=\alpha(x),\alpha(x)\longrightarrow 0$

|$f(x)$ и $\phi(x)$ одного порядка $\Leftrightarrow \exists c_{1},c_{2}>0:$
	$c_{1}|f(x)|\leq|\phi(x)|\leq c_{2}|f(x)|$ или ($\exists \tilde{c}_{1},\tilde{c}_{2}$)
	$\tilde{c}_{1}|\phi(x)|\leq |f(x)|\leq\tilde{c}_{2}|\phi(x)|$

$\Rightarrow) \ \ \exists U(a), \exists c_{1},c_{2}$, что $\forall x\in \dot{U}(a) \ \ |\phi(x)|\leq c_{1}|f(x)|\wedge|f(x)|\leq c_{2}|\phi(x)|\Leftrightarrow \frac{1}{c_{2}}|f(x)|\leq|\phi(x)|$
$\Leftarrow)$ Определение

*Док-во:* 
	по определению предела: $\varepsilon= \frac{|c|}{2} \ \ \exists U(a),\forall x\in \dot{U}(a) \ \ \frac{1}{2}|c|\leq| \frac{f(x)}{\phi(x)}|\leq \frac{3}{2}|c|$ и отсюда по утверждению 
Пример: $3x^{2}$ и $\sin x^{2}$ функции одного порядка при $x\longrightarrow 0$

|Если $f(x)$ при $x\longrightarrow a$ имеет главную часть $A(x-a)^{k}, A\neq0$, то главная часть такого вида определяется однозначно, т.е. $A$ и $k$ зависят  от $f(x)$ и $a$

*Док-во:* От противного
	$f(x)=A(x-a)^{k}+o((x-a)^{k}) \ \ x\longrightarrow a$
	$f(x)=A_{1}(x-a)^{k_{1}}$
1. Пусть $k\neq k_{1} \ k_{1}>k$
	$A(x-a)^{k}+o((x-a)^{k})=A_{1}(x-a)^{k_{1}}+\alpha_{1}(x)(x-a)^{k_{1}}$
	$A+\alpha(x)=A_{1}(x-a)^{k_{1}-k}+\alpha_{1}(x)(x-a)^{k_{1}-k}\implies A=0$
	Т.к. $k_{1}>k\implies k_{1}-k>0\implies(x-a)^{k_{1}-k}\longrightarrow 0\implies A=0$ - невозможно
	Аналогично и с $k_{1}<k\implies A_{1}=0\implies k_{1}=k$
2. Пусть $A=A_{1}:$
	$A(x-a)^{k}+\alpha(x)(x-a)^{k}=A_{1}(x-a)^{k}+\alpha_{1}(x)(x-a)^{k}$
	$A+\alpha(x)=A_{1}+\alpha_{1}(x)\implies A_{1}=A$
