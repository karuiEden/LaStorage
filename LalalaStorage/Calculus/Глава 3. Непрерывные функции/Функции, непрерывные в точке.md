|$f(x)$ на $(a,b), \ \ x_{0}\in(a,b)$ **непрерывна в $x_{0}$**, если $f(x)\longrightarrow f(x_{0}), \ \ x\longrightarrow x_{0}$

|(по Коши) $\forall\varepsilon>0 \ \exists\delta_{\varepsilon}>0: \ \forall x \ \ |x-x_{0}|<\delta_{\varepsilon} \ \ \ |f(x)-f(x_{0})|<\varepsilon$

|(по Гейне) $\forall \{ x_{n} \}\subset(a,b),x_{n}\longrightarrow_{n\longrightarrow \infty}x_{0}, \ \ \ \{ f(x_{n}) \}$ сходится

$\Delta x=x-x_{0}$ - приращение аргумента
$\Delta f=\Delta y=f(x)-f(x_{0})$- приращение функции

|$f(x)$ - разрывная в $x_{0}\in(a,b)$, если либо
1. $f(x)$ не определена в $x_{0}$
2. $f(x)$ в $x_{0}$ определена, но не непрерывна, $x_{0}$ - точка разрыва $f(x)$

|Классификация точек разрыва: пусть $x_{0}$ - точка разрыва
1. если $\exists \lim_{ x \to x_{0}+0 }f(x),\lim_{ x \to x_{0}-0 }f(x)$, то $x_{0}$ - **точка разрыва 1-ого рода**
	$f(x_{0}+0) - f(x_{0}-0)$ - скачок в $x_{0}$
2. если $f(x_{0}+0)=f(x_{0}-0)$, то $x_{0}$ - **точка устранимого разрыва**, если положить $f(x):=\lim_{ x \to x_{0} }f(x)$, то $f(x)$ непрерывна
3. если $x_{0}$ не точка разрыва 1-ого рода, то $x_{0}$ - **точка разрыва 2-ого рода**

Примеры:
1. $sgn(x)$ непрерывна на $\mathbb{R}\setminus \{ 0 \}, \ \ x=0$ - разрыв **1-ого** рода(неустранимый)
2. $sgn^{2}(x)$ устранимый разрыв 1-ого рода
3. $y=\sin \frac{1}{x}, \ \ x\neq 0,\quad x=0$ - ограниченный разрыв 2-ого рода
4. $y= \frac{1}{x}, \ \ x=0$ неограниченный разрыв 2-ого рода
5. Монотонная функция имеет только точки разрыва 1-ого рода

|$f(x)$ на $(a,x_{0}], \ f(x)$ **непрерывна в $x_{0}$ слева**, если $f(x)\longrightarrow f(x_{0})$ при $x\longrightarrow x_{0}-0$

|$f(x)$ непрерывна в $x_{0} \Leftrightarrow f(x)$ непрерывна в $x_{0}$ справа и слева (теорема о пределах)

|$f(x)$ и $g(x)$ непрерывны в $x_{0}\implies \ \ f+g, \ f\times g, \frac{f}{g}$(если $g(x)\neq 0$) непрерывны в $x_{0}$ (арифметические свойства пределов)

|$f(x)$ непрерывна в $x_{0}$ и $f(x_{0})\neq0$, то $\exists U(x_{0})$, в которой $f(x)$ имеет значения одного знака (теорема знаков пределов)

|Пусть $y=\phi(x)$ непрерывна в $x_{0}$, $f(x)$ непрерывна в $y_{0}=\phi(x_{0})\implies f(\phi(x))$ непрерывна в $x_{0}$

*Док-во:*
$\forall\varepsilon>0 \ \exists \eta_{\varepsilon}>0:\forall y \ \ |y-y_{0}|<\eta_{\varepsilon}\implies f(y)$ определена в $y$ и $|f(y)-f(y_{0})|<\varepsilon$
$\exists\delta_{\eta_{\varepsilon}}(=\delta_{\varepsilon})>0 \ \ |x-x_{0}|<\delta_{\varepsilon}\implies|\phi(x)(=y)-\phi(x_{0})(=y_{0})|<\eta_{\varepsilon}$

|$f(y)$ и $\phi(x)$ непрерывны $\implies \ \ f(\phi(x))$ непрерывна в $x_{0}$

|$f(y)$ и $\phi(x)$ непрерывны $\implies \lim_{ x \to x_{0} }f(\phi(x))=f(\lim_{ x \to x_{0} }\phi(x))$
