$f(x)$ рассматривается на $<a,b>$

|$f(x)$ **непрерывна на промежутке $<a,b>$**, если $f(x)$ непрерывна в каждой точке $<a,b>$ относительно $<a,b>$

Если $[a,b]$ - отрезок, то непрерывность в $a$ - непрерывность справа, а в точке $b$ - непрерывностью слева. $\triangleq f(x)\in C(<a,b>)=C<a,b>$

| Первая теорема Больцано-Вейерштрасса: $f(x)\in C[a,b]\implies f(x)$ ограничена на $[a,b]$ ($B[a,b]$)
*Док-во:* от противного
![[Pasted image 20241223192255.png]]
Пусть $f(x)$ не ограничена на $[a,b]$
$\forall n\in \mathbb{N}  \ \ \exists x_{n}\in[a,b]:|f(x_{n})|>n$
$\{ x_{n} \}$ ограничена $\forall n \ \ \ a\leq x_{n}\leq b\implies$ по Б-В $\exists \{ x_{n} \}$ сходится $x_{n_{k}} \longrightarrow x_{0}\in[a,b]$
По определению последовательности Гейне(непрерывности)
$f(x_{n_{k}})\longrightarrow f(x_{0})$ при $k\longrightarrow\infty$. Т.к. $\{ f(x_{n_{k}}) \}$ сходится, то она ограничена $\implies$ противоречие

|Вторая теорема Б-В: Если $f(x)\in C[a,b]$, то $f(x)$ достигает своих точных верхней и нижней граней.
*Док-во:* Для $\max:$
$\exists M:=\sup_{[a,b]}f(x)\in \mathbb{R}, \forall n\in \mathbb{N} \ \ \exists x_{n}\in[a,b]:f(x_{n})>M- \frac{1}{n}$
$\{ x_{n} \}$ ограничена $\implies$ по Б-В $\exists \{ x_{n_{k}} \}$ сходится $x_{n_{k}}\longrightarrow x_{0}(k\longrightarrow\infty)$ 
$f(x)$ непрерывна в $x_{0}$ по определению непрерывности по Гейне:
$f(x_{n_{k}})\longrightarrow f(x_{0}), \ \ f(x_{n_{k}})\longrightarrow f(x_{0})<M$
$M- \frac{1}{k}\leq M - \frac{1}{n_{k}}<f(x_{n_{k}}); \ \ M - \frac{1}{k}\longrightarrow M \ \ (k\longrightarrow\infty)$
По теореме о зажатой последовательности: $f(x_{0})=M$

|$f(x)\in C[a,b]$ Если $f(a)\cdot f(b)<0$, то $\exists c\in[a,b]$ при котором $f(c)=0$
*Док-во:*
$\Delta_{0}:=[a,b]=[a_{0},b_{0}], \ c_{0}:=\frac{a_{0}+b_{0}}{2}$
Если $f(c_{0})=0\implies \ c$ найдена
	$f(c_{0})\neq0,$ то $\Delta_{1}$- тот из $[a_{0},c_{0}]$ и $[c_{0},b_{0}]$, на концах которого значения разных знаков, $\Delta_{1}\subset\Delta_{0}$
	$\Delta_{1}:=[a_{1},b_{1}], \ c_{1}:=\frac{a_{1}+b_{1}}{2}$
$\Delta_{2}$ - тот из $[a_{1},c_{1}]$ и $[c_{1},b_{1}]$, на концах которого значения разных знаков и т.д.
1. при некоторых $n \ \  f(c_{n})=0$
2. при всех $n$ $f(c_{n})\neq0:\Delta_{0}\supset\Delta_{1}\supset\Delta_{2}\supset\dots \supset\Delta_{n}: \{ \Delta_{n} \}$ стягивается $\implies$ по полноте Кантора $\exists!\Delta_{n}=c \in[a_{n},b_{n}] \ \ \forall n$
	На концах которого значения разных знаков:
	$f(c)=\lim_{ n \to \infty }f(a_{n})=\lim_{ n \to \infty }f(b_{n})$
$x_{n}' \ a_{n}$ или $b_{n}$, но $f(x_{n}')<0$
$x_{n}'' \ \ a_{n}$ или $b_{n}$, в которой $f(x_{n}'')>0,\lim_{ n \to \infty }f(x_{n}')\leq0, \ \lim_{ n \to \infty }f(x_{n}'')\geq0\implies f(c)=\lim_{ n \to \infty }f(x_{n}')=\lim_{ n \to \infty }f(x_{n}'')\implies f(c)=0$

|Следствие: $f(x)\in C[a,b], \ \ f(a)=A, \ f(b)=B$
	Тогда $\forall C$ между $A$ и $B$ $\exists c\in[a,b]: \ f(c)=C$
*Док-во:* $\phi(x)=f(x)-C$

|$f(x)\in C[a,b]$ и строго возрастает на $[a,b], \ \ A:=f(a),B:=f(b)$
	Тогда $f([a,b])=[A,B]$ и обратная к  $y=f(x)$ функция $x=\phi(y)$
1. Однозначна на $[A,B]$
2. $\phi(y)\in C[A,B]$
3. $\phi(y)$ строго возрастает.
*Док-во:*
1. $f(x)$ возрастает на $[a,b]\implies \forall x \in[a,b]$
	$a\leq x\leq b\implies f(a)=A\leq f(x)\leq B=f(b)\implies f(x)\in[A,B]$
	По теореме  о промежуточном значении:
	$\forall y\in[A,B] \ \ \exists x \in[a,b]:y=f(x),$ т.е. $f([a,b])=[A,B]$ 
2. Однозначность обратной функции. Если не так $\exists x_{1},x_{2}\in[a,b], \ x_{1}\neq x_{2}; f(x_{1})=f(x_{2})\implies$ противоречие строгой монотонности
3. Обратная $\phi(y)$ строго возрастает на $[A,B]$. Если не так, то $\exists y_{1},y_{2}\in[A,B], \ \ y_{1}<y_{2}: \ \phi(y_{1})\geq \phi(y_{2})$
	"=": невозможно(однозначность $\phi(y)$)
	">": $x_{1}>x_{2}$, то $f(x_{1})=y_{1}<y_{2}=f(x_{2})$, противоречит строгому возрастанию
4. $\phi(y)\in C[A,B]$ Непосредственная проверка
	1. $y_{0}\in[A,B]$
	![[Pasted image 20241223211448.png]]
		$\forall\varepsilon>0$, такое что
		$(x_{0}-\varepsilon,x_{0}+\varepsilon)\subset[a,b]$
		$y_{1}:=f(x_{0}-\varepsilon)=f(x_{1})$
		$y_{2}=f(x_{0}+\varepsilon)=f(x_{2})$
		$f(x_{1})<f(x_{0})<f(x_{2})$
		$\delta_{\varepsilon}:=\min\{ y_{2}-y_{0},y_{0}-y_{1} \}$
		$(y_{0}-\delta_{\varepsilon},y_{0}+\delta_{\varepsilon})\subset(y_{1},y_{2})$
		$\forall y:|y-y_{0}|<\delta_{\varepsilon}$
		$y\in(y_{0}-\delta_{\varepsilon},y_{0}+\delta_{\varepsilon})$
		$\phi(y_{0})-\varepsilon=x_{1}=\phi(y_{1})\leq\phi(y_{0}-\varepsilon)<\phi(y)<\phi(y_{0}+\varepsilon)\leq \phi(y_{2})=x_{2}=\phi(y_{0})+\varepsilon$
		$\forall y:|y-y_{0}|<\delta_{\varepsilon}\implies|\phi(y)-\phi(y_{0})|<\varepsilon$
	2. $y_{0}=A\vee y_{0}=B$
	![[Pasted image 20241223212141.png]]
		$\forall\varepsilon:a+\varepsilon \in([a,b])$
		$\delta_{\varepsilon}:=f(a+\varepsilon)-f(a)>0$
|Критерий непрерывности монотонной функции: $f(x)$ монотонна на $<a,b>$, $f(x)\in C<a,b>\Leftrightarrow f(<a,b>)$ - промежуток
*Док-во:* 
$\Rightarrow$) если $f(x)\in C<a,b>$, то  независимо от монотонности $f(<a,b>)$ промежуток
$\Leftarrow$)  От противного: Пусть $f(x)\nearrow$ , $f(<a,b>)$   - промежуток, то $f(x)$ не непрерывна на $<a,b>$ , т.е. $\exists x_{0}\in<a,b>$ - точка разрыва 1-ого рода
$f(x_{0}-0)\leq f(x_{0})\leq f(x_{0}+0)$ - хотя бы одно строгое неравенство
![[Pasted image 20241223212849.png]]
$(f(x_{0}-0),f(x_{0}+0)) \quad f(x_{0}-0)<f(x_{0}+0), \exists A\in(f(x_{0}-0),f(x_{0}+0))$
$f(x_{0}-0)<A<f(x_{0}+0)$
При всех $x<x_{0}\quad f(x)\leq f(x_{0}-0)<A$ 
		$x>x_{0}\quad f(x)\geq f(x_{0}+0)>A\implies$
			$f(<a,b>)$ не содержит $A\implies f(<a,b>)$ не промежуток

|Критерий однозначной обратимости непрерывной функции: Пусть $f(x)\in C[a,b]$, тогда $f(x)$ имеет однозначную обратную функцию $\Leftrightarrow$ $f(x)$ строго монотонна
*Док-во:*
$\Leftarrow$) По теореме о промежуточном значении
$\Rightarrow$) $Y=f([a,b])  \ \ \forall y\in Y \ \ \exists!x \in[a,b]: f(x)=y\implies f(a)\neq f(b)$. Для определения $f(a)<f(b)$. При таком предположении докажем строгое возрастание.
1. $f(a)=\min f(x), \ f(b)=\max f(x)$ на $[a,b]$. Пусть не так, т.е. $\exists x_{0}\in(a,b)$
	$f(a)>f(x_{0}),f(x_{0})<f(a)<f(b)$
	На $[x_{0},b] \ \ \exists x'\in[x_{0},b]:f(x')=f(a)$
	$\exists x':f(x')=f(a)$ противоречие однозначной обратимости.
	$f(b)=\max f(x)$ на $[a,x_{0}]$ аналогично
	![[Pasted image 20241223214929.png]]
1. Проверка строгого возрастания
	От противного: пусть $f(x)$ нестрого возрастает
	$\exists x',x'' \ x'< x'' \wedge f(x')\geq f(x'')$, "=" невозможно,
	$x'<x''$, но $f(x')>f(x'')$
	$f(a)<f(x'')<f(x')$ на $[a,x']$, тогда по т. о промежуточных значениях: $\exists c \ \ f(c)=f(x''), \ c\neq x''$ - нарушает однозначную обратимость.
