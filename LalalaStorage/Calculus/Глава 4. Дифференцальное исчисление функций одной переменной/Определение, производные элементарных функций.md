| $y=f(x): U(a), \ x\in \dot{U}(a)$
	$\frac{f(x)-f(a)}{x-a}=\frac{\Delta y}{\Delta x}$, где $x-a$ - разностное отношение, $\Delta y$ - приращение функции, $\Delta x$ - приращение аргумента.
|Если $\exists \lim_{ x \to a } \frac{f(x)-f(a)}{x-a}$, то говорят, что $f(x)$ имеет в $a$ **производную**: $f'(a),\ y', \ \frac{df}{dx}$

|$y=f(x)$ в правой(левой) окрестности $a$($[a,b)\vee(c,a]$) $\exists \lim_{ x \to a+0 } \frac{f(x)-f(a)}{x-a}$, то существует правая(левая) производная в $a$.
$\triangleq f'_{+}(a), \ f'_{-}(a)$
Примеры:
1. $y=C\implies\Delta y=0\implies C'=0$
2. $y=\sin x$
	$\Delta y=\sin(x+\Delta x)-\sin x=2\sin \frac{\Delta x}{2}\cos (x+\frac{\Delta x}{2})$
	$\frac{\Delta y}{\Delta x}=2\sin \frac{\Delta x}{2} \frac{\left( \cos \left( \frac{\Delta x}{2} \right)+x \right)}{\Delta x}=\cos\left( x+ \frac{\Delta x}{2}  \right)\cdot \frac{\sin\left(  \frac{\Delta x}{2}  \right)}{ \frac{1}{2}\Delta x }\longrightarrow_{\Delta x\longrightarrow 0}\cos x$
	$\sin' x=\cos x=\sin\left( x+ \frac{\pi}{2} \right)$
3. $y=\cos x$
	$\Delta y=\cos(x+\Delta x)-\cos x=-2\sin\left(  \frac{\Delta x}{2} \right)\sin\left( x + \frac{\Delta x}{2}  \right)$
	$\frac{\Delta y}{\Delta x}=-2\sin\left( x+ \frac{\Delta x}{2} \right) \frac{\sin\left( \frac{\Delta x}{2} \right)}{\Delta x}=-\sin\left( x+\frac{\Delta x}{2} \right) \frac{\sin \frac{\Delta x}{2}}{\frac{1}{2}\Delta x}\longrightarrow_{\Delta x\longrightarrow 0}-\sin x$	
4. $y=e^{ x }$
	$\Delta y=e^{ x+\Delta x }-e^{ x }=e^{ x }(e^{ \Delta x }-1)=e^{ x }(\Delta x+o(\Delta x)) \ \ \Delta x\longrightarrow 0$
	$\frac{\Delta y}{\Delta x}=e^{ x } \frac{\Delta x+o(\Delta x)}{\Delta x}=e^{ x }(1+o(1))\longrightarrow_{\Delta x\longrightarrow 0}e^{ x }$
	$(e^{ x })'=e^{ x }$
5. $y=x^{a}$
	1.  $x$ такой, что $x^{a}$ определена $a\in \mathbb{R}, \ x>0$
	$\Delta y=(x +\Delta x)^{a}-x^{a}=x^{a}\left[ \left( 1+ \frac{\Delta x}{x} \right)^{a} - 1 \right]=x^a\left[ a\cdot \frac{\Delta x}{x} +o(\Delta x) \right]=x^{a-1}[a+o(1)]\Delta x$
	$\frac{\Delta y}{\Delta x}=x^{a-1}[a+o(1)]\longrightarrow_{\Delta x\longrightarrow 0}ax^{a-1}$
	2. $x=0$
		1. $a>1 \ \ \ \frac{\Delta y}{\Delta x}=\frac{\Delta x^{a}}{\Delta x}=\Delta x^{a-1}\longrightarrow 0$
			$(x^{a})'=ax^{a-1}|_{x=0}=0$
		2. $a=1 \ \ \frac{\Delta y}{\Delta x}= \frac{\Delta x}{\Delta x}=1$
		3. $a<1 \ \ \ \frac{\Delta y}{\Delta x}= \frac{\Delta x^{a}}{\Delta x}= \frac{1}{\Delta x^{1-a}}\longrightarrow\infty$
			$(x^{a})'=ax^{1-a} \ \ \ x\geq0$
6. $y=|x|$ в $a=0 \ \ \ y'_{+}(0)=1 \ \ y'_{-}(0)=-1$
