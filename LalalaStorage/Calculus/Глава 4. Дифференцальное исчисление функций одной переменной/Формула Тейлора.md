$P_{n}(x)=p_{0}+p_{1}x+p_{2}x^{2}+\dots+p_{n}x^{n}, p_{n}=0$
$P_{n}(0)=p_{0}$
$P_{n}'(x)=p_{1}+2p_{2}x+\dots np^{n-1}\implies p'_{n}(0)=p_{1}$
$P''_{n}(x)=2p_{2}+3\cdot 2p_{3}x+\dots+n(n-1)p_{n}x^{n-2}\implies p''_{n}(0)=2!p_{2}$
$P_{n}^{(n)}(x)=p_{n}^{(n)}(0)=n!p_{n}\implies \forall k=0,\dots,n \ \ \ p_{k}= \frac{p_{n}^{(k)}(0)}{k!}$
$$P_{n}(x)=p_{n}(0)+p_{n}'(0)x+\frac{p_{n}''(0)}{2!}x^{2}+\dots+ \frac{p_{n}^{(n)}(0)}{n!}x^{n}$$
*Задача:* Записать $P_{n}(x)$ по степеням $(x-a)$
	$P_{n}(x)=p_{0}+p_{1}x+p_{2}x^{2}+\dots+p_{n}x^{n}=p_{0}+p_{1}[(x-a)+a]+p_{2}[(x-a)+a]^{2}+\dots+p_{n}[(x-a)+a]^{n}\implies$ записать так можно.
$P_{n}(x)=\tilde{p_{0}}+\tilde{p_{1}}(x-a)+\dots+\tilde{p_{n}}(x-a)^{n}\quad \tilde{p_{n}}\neq 0(p_{n}\neq 0)$ при $x=a \ \ \ \ \tilde{p_{0}}=P_{n}(a)$
Дифференцируем и подставляем $x=a:$
$\tilde{p_{k}}= \frac{p^{(k)}_{n}(a)}{k!}\quad k=0,\dots,n \quad \forall a:$
$$P_{n}(x)=p_{n}(a)+ \frac{p_{n}'(a)}{1!}(x-a)+\dots+\frac{p^{(n)}_{n}(a)}{n!}(x-a)^{n}$$
|Многочлен Тейлора порядка $n$ для $f(x)$ в $a$:
	$f(x)$ имеет в $x=a$ $n$ производных
	$f(x)\thicksim p_{n}(x,a):=f(a)+ \frac{f'(a)}{1!}(x-a)+\dots+\frac{f^{(n)}(a)}{n!}(x-a)^{n}$

|$r_{n}(x,a):=f(x)-p_{n}(x,a)$ - $n$-остаточный член формулы Тейлора в $a$

$f(x)=p_{n}(x,a)+r_{n}(x,a)$

$r^{(k)}_{n}(x,a)|_{x=a}=0\quad k=0,1,\dots,n$
$r_{n}(x,a)=f(x)-\left[ f(a)+\frac{f'(a)}{1!}(x-a) + \frac{f''(a)}{2!}(x-a)^{2}+\dots+ \frac{f^{(n)}(a)}{n!}(x-a)^{n} \right]$
$r_{n}(x,a)|_{x=a}=f(x)|_{x=a}-f(a)=0$
$r_{n}'(x,a)=f'(x)-\left[ f'(a)+\frac{f''(a)}{1!}(x-a)+\dots+\frac{f^{(n)}(a)}{(n-1)!}(x-a)^{n-1} \right]$
$r'_{n}(x,a)|_{x=a}=f'(x)|_{x=a}-f'(a)=0$

$\phi(x):=r_{n-1}(x-a) \quad \phi(a)=\phi'(a)=\dots=\phi^{(n-1)}(a)=0$
$\psi(x):=(x-a)^{n}\quad \psi(a)=\psi'(a)=\dots=\psi^{(n-1)}(a)=0$  
Всё это приводит к утверждению из §7

|Остаток в форме Лагранжа: $f(x)\in C[a,b], \ \ n-1$ раз дифференцируем на $[a,b), n$ раз дифференцируем на $(a,b)$. Тогда $\forall x\in[a,b] \ \ \exists\xi=\xi_{x}\in(a,x):$
$f(x)=\sum_{k=0}^{n-1} \frac{f^{(k)}(a)}{k!}(x-a)^{k}+ \frac{f^{(n)}(\xi)}{n!}(x-a)^n$

$$\frac{f^{(n)}(\xi)}{n!}(x-a)^{n}\text{ - остаток в форме Лагранжа}$$
*Док-во:* Применение формулы Коши для $n$ производных к паре $\phi(x)=r_{n-1}(x,a), \ \psi(x)=(x-a)^{n}\quad \exists\xi \in(a,x)$
$$\frac{r_{n-1}(x,a)}{(x-a)^{n}}=\frac{r^{(n)}_{n-1}(x,a)|_{x=\xi}}{n!}\implies r_{n-1}(x,a)=\frac{1}{n!}r_{n-1}^{(n)}(\xi,a)(x-a)^{n}=$$
$$=\frac{1}{n!}\left( f(x)-\sum_{k=0}^{n-1} \frac{f^{(k)}(a)}{k!}(x-a)^{k} \right)^{(n)}|_{x=\xi}(x-a)^{n}=\frac{1}{n!}f^{(n)}(\xi)(x-a)^{n}$$
|Следствие: Если при некотором $n$ $f^{(n)}(x)=0\implies f(x)$ - многочлен степени не выше $n$

|Остаток в форме Пеано: $f(x)$ $n$ раз дифференцируема в $a\implies$
$$f(x)=\sum_{k=0}^{n} \frac{f^{(k)}(a)}{k!}(x-a)^{k}+o((x-a)^{n})\quad x\longrightarrow a$$
Остаток выглядит:
$$o((x-a)^{n})\quad x\longrightarrow a$$
*Док-во:* $f^{(n)}_{(a)}$ существует в $a\implies f^{(n-1)}(x)$ определена в $U(a)$
$\phi(x):=r_{n-1}(x,a)\ \ \ \psi(x):=(x-a)^{n} \ \ \ \exists\xi$ по утверждению типа Коши
$$\frac{r_{n-1}(x,a)}{(x-a)^n}=\frac{r_{n-1}^{(n-1)}(\xi,a)}{n!(\xi-a)}=$$
$$\frac{\left( \left[ f(x)-\sum_{k=0}^{n-1}f^{(k)}(a)(x-a)^{k}\right]^{n-1}|_{x=\xi} \right)}{n!(\xi-a)}=\frac{1}{n!} \frac{f^{(n-1)}(\xi)-f^{(n-1)}(a)}{\xi-a}=\frac{1}{n!}(f^{(n)}(a)+o(1))\quad x\longrightarrow a$$
$$r_{n-1}(x,a)=\frac{1}{n!}f^{(n)}(a)(x-a)^{n}+o((x-a)^{n})(=r_{n}(x,a))$$
