**Определение**: Пусть $p$ - разбиение $[a,b]$, $\xi \in V_{p}$ 
$$\sigma(f,p,\xi):=\sum_{i=1}^{n}f(\xi_{i})\cdot\Delta x_{i}$$
	называется **интегральной суммой Римана** для $f$, соответствующей разбиению $p$ и выборке $\xi$ 

**Определение:** Если $\exists I\in \mathbb{R}:\forall\varepsilon>0 \ \ \exists\delta>0  \ \ \forall p \in P_{\delta} \ \ \forall\xi \in V_{p} \ \ \ \ \ |\sigma(f,p,\xi)-I|<\varepsilon$, то $f$ интегрируема по Риману на $[a,b]$, а число $I$ называется **интегралом Римана** $f$ по $[a,b]$.
*Обозначение: интегрируемости по Риману* $f\in \mathcal{R}[a,b]$. 
*Обозначение:* $I=\int_{a}^{b}f(x)dx:=\lim_{  L(p) \to 0 }\sigma(f,p,\xi)$  

**Определение в терминах сумм Дарбу:** $f: [a,b]\to \mathbb{R}$ интегрируема по Риману, если $\exists I\in \mathbb{R}: \forall\varepsilon>0 \ \ \exists \delta(\varepsilon)>0: \forall\text{разбиения } P\  [a,b] \ L(p) < \delta(\varepsilon):$
$$\begin{cases}
|s(f,P) - I| <\varepsilon \\
|S(f,P)-I|<\varepsilon
\end{cases}$$
**Замечание:** Число $I$ однозначно определено.
*Док-во:* Если $J=\int_{a}^{b}f(x)dx$
$$|I-J|\leq|I-\sigma(f,P,\xi)|(<\varepsilon)+|\sigma(f,P,\xi)-J|(<\varepsilon)\implies|I-J|=0$$ 
*Пример функции, которая не интегрируема по Риману:*
$$f(x)=\begin{cases}
1, \ x\in \mathbb{Q} \cap[0,1] \\
0,x \in \mathbb{R}\setminus \mathbb{Q} \cap[0,1]
\end{cases}$$
$f\not\in \mathcal{R}([0,1])$
Пусть $T$ разбиение $[0,1]$
$$\begin{cases}
s(f,T)=0 \\
S(f,T)=1
\end{cases}\implies \exists\varepsilon=1:\forall\delta(\varepsilon)\forall \ \ P[0,1]\to|s(f,P)-S(f,P)|\geq 1\implies \not\exists I\in \mathbb{R} \text{ по условию}$$

**Критерий Коши интегрируемости по Риману:** $f\in \mathcal{R}[a,b]\Leftrightarrow\forall\varepsilon>0 \ \exists\delta>0 \ \forall p,q\in P_{\delta}, \ \forall\xi \in V_{p}, \ \forall\eta \in V_{q} \ \ \ |\sigma(f,p,\xi)-\sigma (f,q,\eta)|<\varepsilon$ 

**Теорема:** (Необходимое условие интегрируемости по Риману)
Пусть $f(x)\in \mathcal{R}([a,b])$, тогда $f(x)$ ограничена на $[a,b]$
*Док-во:*
По определению интегрируемости найдётся разбиение $T$ $[a,b]$, что $$\begin{cases}
s(f,T)\in \mathbb{R} \\
S(f,T)\in \mathbb{R}
\end{cases}\implies \ \text{по лемме} f(x) \text{ ограничена снизу и сверху}\implies f(x) \text{ ограничена на } [a,b]$$
**Лемма**: Пусть $f:[a,b]\to \mathbb{R}$. Тогда $\forall T\ [a,b] \ \ \ s(f,T)\leq S(f,T)$
**Лемма:** Пусть $f:[a,b]\to \mathbb{R}$ и $T_{1},T_{2}$ разбиения $[a,b]: T_{2}\succcurlyeq T_{1}$
Тогда $s(f,T_{2})\geq s(f,T_{1}) \ \ \ \ \ S(f,T_{2})\leq S(f,T_{1})$

*Док-во:* $T_{2} \succcurlyeq T_{1}\implies T_{2}$ содержит все точки $T_{1}$ и возможно ещё какие-то точки.
Рассмотрим сначала нижние суммы Дарбу
Считаем сначала, что $T_{2},T_{1}$ отличаются на одну точку
![[Pasted image 20250313111026.png]]
$$s(f,T_{2})-s(f,T_{1})=\inf_{x\in[x_{i-1},c]}f(x)(c-x_{i-1})+\inf_{x\in[c, x_{i}]}f(x)(x_{i}-c)-\inf_{x\in[x_{i-1},x_{i}]}f(x)(x_{i}-x_{i-1})\geq$$
$$\begin{array} \\
\inf_{x\in[x_{i-{1}},c]}f(x) \geq \inf_{x\in[x_{i-{1}},x_{i}]}f(x) \\
\inf_{x\in[c,x_{i}]}f(x) \geq \inf_{x\in[x_{i-{1}},x_{i}]}f(x) \\
\end{array}$$
$$\geq \inf_{x\in[x_{i-1},x_{i}]}f(x)(c-x_{i-1})+\inf_{x\in[x_{i-1}, x_{i}]}f(x)(x_{i}-c)-\inf_{x\in[x_{i-1},x_{i}]}f(x)(x_{i}-x_{i-1})=0$$
$T_{2}\succcurlyeq T_{1}\implies \exists$ набор упорядоченных разбиений $T_{2}=T^{k}\succcurlyeq T^{k-1}\succcurlyeq\dots\succcurlyeq T^{0}=T_{1}$, где $T^{j}$ отличается от $T^{j-1}$ добавлением одной точки на интервал
$$s(f,T^{k})\geq s(f,T^{k-1})\geq\dots\geq s(f,T^{0})$$
Для верхних сумм док-во аналогичное.

**Следствие**: $f:[a,b]\to \mathbb{R}$
Пусть $T_{1},T_{2}$ - произвольные разбиения $[a,b]$. Тогда $s(f,T_{1})\leq S(f,T_{2})$
*Док-во*: Рассмотрим разбиение $T:= T_{1}\cup T_{2}$, где $T\succcurlyeq T_{1}$ и $T\succcurlyeq T_{2}$ . По предыдущей лемме
$$\begin{array}
S(f,T_{2})\geq S(f,T) \\
s(f,T)\geq s(f,T_{1})
\end{array}\implies s(f,T_{1})\leq S(f,T_{2})
$$
