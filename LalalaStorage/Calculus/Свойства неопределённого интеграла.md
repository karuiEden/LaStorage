1. Линейность
	**Теорема:** Пусть $-\infty <a<b<+\infty$ и $\exists F$ для $f_{1}$ на $(a,b)$ и для $f_{2}$ на $(a,b)$.
		Тогда при $|\alpha_{1}|+|\alpha_{2}|>0$ $\exists F$ для $\alpha_{1}f_{1}+\alpha_{2}f_{2}$ и более того $\int[\alpha_{1}f_{1}(x)+\alpha_{2}f_{2}(x)]dx=\alpha_{1}\int f_{1}(x)dx+\alpha_{2}\int f_{2}(x)dx$ 
	*Док-во:* Пусть $F_{1}$ - первообразная для $f_{1}$, а $F_{2}$ - первообразная для $f_{2}$.
		$\implies\alpha_{1}F_{1}+\alpha_{2}F_{2}$ - первообразная для $\alpha_{1}f_{1}+\alpha_{2}f_{2}\implies$
		$\int[\alpha_{1}f_{1}(x)+\alpha_{2}f_{2}(x)]dx=\alpha_{1}F_{1}(x)+\alpha_{2}F_{2}(x)+C$
		С другой стороны $\alpha_{1}\int f_{1}(x)dx=\alpha_{1}F_{1}(x)+C_{1}, \ \ \alpha_{2}\int f_{2}(x)dx=\alpha_{2}F_{2}(x)+C_{2}$
		Тогда $\alpha_{1}F_{1}(x)+\alpha_{2}F_{2}(x)+C_{1}+C_{2}=\alpha_{1}F_{1}(x)+\alpha_{2}F_{2}(x)+C, C=C_{1}+C_{2}$ 
		Равенство надо понимать как равенство семейств функций.
