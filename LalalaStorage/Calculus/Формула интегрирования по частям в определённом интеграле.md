$u,v\in C^{1}[a,b]$(непрерывно дифференцируемы, т.е. $\exists u',v'$ на $[a,b]$ и $u',v'\in C[a,b]$). Тогда $$\int_{a}^{b}u(x)v'(x)dx=u(x)v(x)\bigg|_{a}^{b}-\int_{a}^{b}v(x)u'(x)dx$$
*Док-во:* $(u\cdot v)$ является первообразной для $uv'+u'v$ на $[a,b]$ 
По формуле Н-Л
$$\int_{a}^{b}[u(x)v'(x)dx+u'(x)v(x)]dx=(u\cdot v)(b)-(u\cdot v)(a)=(u\cdot v)\bigg|_{a}^{b}$$
$\implies$ в силу линейности интеграла Римана
$$\int_{a}^{b}u(x)v'(x)dx=U(x)V(x)\bigg|_{a}^{b}-\int_{a}^{b}v(x)u'(x)dx$$



**Пример:** $\int_{0}^{1}\arcsin xdx$
$$u=\arcsin x \ \ du=\frac{dx}{\sqrt{ 1-x^{2} }}, \ \ u \not\in C^{1}[0,1], dv=dx, \ \ v=x$$
$$u\in C^{1}[0,1)$$, проинтегрируем на отрезке $[0,1-\varepsilon]$, $$\int_{0}^{1-\varepsilon}\arcsin xdx=x\arcsin x\bigg|_{0}^{1-\varepsilon}-\int_{0}^{1-\varepsilon} \frac{xdx}{\sqrt{ 1-x^{2} }}=x\arcsin x\bigg|_{0}^{1-\varepsilon}+\sqrt{ 1-x^{2} }\bigg|_{0}^{1-\varepsilon}$$
$$\lim_{ \varepsilon \to 0 } \int_{0}^{1-\varepsilon}\arcsin xdx=\int_{0}^{1}\arcsin xdx=\frac{\pi}{2}-1$$
