**Определение:** Функцией $u=f(x_{1},x_{2},\dots,x_{n})$ $n$ переменных, определённой на $G\subset \mathbb{R}^{n}$ назывют отображение $f:G_{f}\to \mathbb{R}$, ставящее в соответствие каждой точке $x=(x_{1},x_{2},\dots,x_{n})\in G_{f}$ единственное число $u$.

**Определение:** Число $A$ называется пределом функции $u=f(x_{1},x_{2},\dots,x_{n})$ в точке $a=(a_{1},a_{2},\dots,a_{n})$, $A=\lim_{ x \to a }f(x)$
По Коши:
$$\forall\varepsilon>0 \ \exists\delta>0: \forall x\in X\subset \mathbb{R}^{n} \ \|x-a\|<\delta, x\neq a \to \ \ |f(x)-A|<\varepsilon$$
По Гейне:
$$f(x):X\subset \mathbb{R}^{n}\to \mathbb{R}$$

$A$ предел $f(x)$ при $x\to x^{0},x\in X$, если $\forall \{ x^{k} \}_{k=1}^{\infty}\subset X,x^{k}\neq x^{0} \ \forall k \  \{ f(x^{k}) \}$ сходится к $A$ 

$f(x)\to A$ при $x\to x^{0}, x\in X$ 
$$A=\lim_{ x \to x^{0}, x\in X } f(x) \text{ - предел в }x^{0}\text{ вдоль множества }X\text{ или по }X$$
Определениея по Коши и Гейне эквиваленты.
**Определение:** $f:X\subset \mathbb{R}^{n}\to \mathbb{R} \ \ f(x)$ ограничена, если $\exists M>0:\forall x\in X \ \ |f(x)|<M$ 

**Критерий Коши:** $f(x)$ в $\dot{U}(a)$ $\exists \lim_{ x \to a }f(x)\in \mathbb{R}\Leftrightarrow\forall\varepsilon>0 \ \exists\delta>0:\forall x',x''\in \dot{U}_{\delta(a)} \ \ |f(x')-f(x'')|<\varepsilon$

$$\lim_{ (x,y)\to (x_{0},y_{0}) } f(x,y)\text{ - двойной предел}$$
$$\lim_{ x \to x_{0} } (\lim_{ y \to y_{0} }f(x,y) ) - \text{повторный предел}$$
$$\lim_{ y \to y_{0} } \lim_{ x \to x_{0} } f(x,y)$$
**Теорема:** $\lim_{ (x,y) \to (x_{0},y_{0}) }f(x,y)=\lim_{ x \to x_{0} }\lim_{ y \to y_{0} }$ Двойной предел равен повторному, если существует и существует внутренний предел, следовательно существует и равенство.
