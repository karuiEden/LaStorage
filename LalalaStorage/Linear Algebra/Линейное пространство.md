
Множество $V$, состоящее из некоторых элементов $\{ x_{k} \}$ называется линейным пространством над множеством $K$ ($K=\mathbb{R}$ или $K=\mathbb{C}$ ), если для элементов этого множества определены операция сложения этих элементов и умножение элемента на число из множества $K$, такие что ($\forall x_{1}, x_{2} \in V$ и $\forall \lambda \in K$):
$$а)\quad x_{1} + x_{2} \in V$$
$$б) \quad  \lambda \cdot x_{1} \in V$$

для которых выполнены следующие *аксиомы линейного пространства*:
1. $x_{1} + x_{2} = x_{2} + x_{1}$  (коммутативность)
2. $(x_{1} + x_{2}) + x_{3} = x_{1} + (x_{2} + x_{3})$ (ассоциативность по сложению)
3. $\exists \theta \in V, \ x_{1} + \theta = x_{1}$ (существование нулевого элемента)
4. $\exists x_{1}' \ x_{1} + x_{1}' = \theta$ (существование противоположного элемента)
5. $\lambda(x_{1} + x_{2}) = \lambda x_{1} + \lambda x_{2}$ (дистрибутивность)
6. $(\lambda_{1} + \lambda_{2})x = \lambda_{1}x+\lambda_{2}x$ (дистрибутивность)
7. $(\lambda_{1} \cdot \lambda_{2})x = \lambda_{1}(\lambda_{2}x)$ (ассоциативность по умножению)
8. $1\cdot x = x$ (аксиома унитарности)
Иногда элементами линейного пространства называют **векторами**.

*Теорема:* Множество всех матриц размерности $m\times n$ являются линейным пространством над $\mathbb{R}$. 
*Док-во:* доказывается через проверку аксиом л.п.
