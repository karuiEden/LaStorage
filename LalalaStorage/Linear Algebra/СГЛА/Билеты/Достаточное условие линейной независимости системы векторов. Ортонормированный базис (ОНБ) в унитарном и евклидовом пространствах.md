**Определение:** $x,y\in U(E)$ называются *ортогональными*, если $(x,y)=0$
Обозначение: $x\perp y$ 

**Теорема 2(Достаточные условия Л.Н. системы ненулевых векторов):** 
$a_{i}\in U(E); \ i=1,k$
$a_{i}\neq\theta$ и $a_{i}\perp a_{j}$ при $i\neq j \ \ i,j = \overline{1,k}$
Тогда система $a_{1},a_{2},\dots,a_{k}$ - Л.Н. система
$\blacktriangleleft$
 Рассмотрим $\lambda_{1}a_{1}+\dots+\lambda_{k}a_{k}\  ((,a_{i}))=\theta \ ((, a_{i}))$
$$\lambda_{1} (a_{1},a_{i})(=0)+\dots+\lambda(a_{k},a_{i})(=0)=(\theta,a_{i})=0$$


Получаем $\lambda_{i}\|a_{i}\|^{2}(\neq 0)=0\implies\lambda_{i}=0; \ i= \overline{1,k}$ 
Т. е. $a_{1},\dots ,a_{k}$ - ЛН(Линейно независимы) 
$\blacktriangleright$

**Определение:** Система $a_{1},\dots ,a_{k}$ называется *ортогональной*, если $a_{i}\perp a_{j};\ i\neq j;\ i,j= \overline{1,k}$

**Определение:** Система $a_{1},\dots,a_{k}$ называется *ортонормированной*(ОНС), если $(a_{i},a_{j})=\delta_{ij}; \ i,j=\overline{1,k}$ 

**Замечание:** $a_{1},\dots,a_{k}$ - ОНС $\xrightarrow{T{2}}a_{1},\dots,a_{k}$ - Л.Н.

**Замечание:** Если $\dim(U(E))=n$ и $a_{1},\dots,a_{n}$ - ОНС $\implies$ $a_{1},\dots,a_{n}$ - базис в $U(E)$ 

**Определение:** Базис $U(E)$ $[e]$ называется *ортонормированным базисом (ОНБ)*, если $$(e_{i},e_{j})=\delta_{ij}; \ i,j= \overline{1,n} \ (n=\dim U(E))$$
**Замечание** Вид $(x,y)$ в ОНБ:
$$(x,y)=x^{T}_{\downarrow}G_{a}\overline{y}_{\downarrow}=x^{T}_{\downarrow}E\overline{y}_{\downarrow}=x^{T}_{\downarrow}\overline{y}_{\downarrow}=\sum_{i=1}^{n}x_{i}\overline{y}_{i}$$ 
$$g_{ij}=(e_{i},e_{j})=\delta_{ij}(i,j =\overline{1,n})=G_{e}=E$$

$$(x,y)=\sum_{i=1}^{n}x_{i}y_{i}=x^{T}_{\downarrow}y_{\downarrow}\text{ в }E$$

**Теорема 3($\exists$ОНБ в $n$-мерном $U(E)$):** Пусть $\dim U(E)=n, n\in \mathbb{N}$.
Тогда $\exists$ ОНБ $U(E)$

$\blacktriangleleft$
$\dim U=n\implies \exists f_{1},f_{2},\dots,f_{n}$ - базис $U$
Докажем по индукции утверждение
	**Утверждение:** Пусть $\exists$ $e_{1},\dots,e_{k}$ - ОНС, таких, что они являются линейной комбинацией векторов $f_{1},\dots,f_{k}$.
	Тогда $\exists e_{k+1}:\|e_{k+1}\|=1$ и $e_{k+1}\perp e_{i}, \ i=\overline{1,k}$ .

1. $k=1:$ $g_{1}=f_{1}; \ e_{1}= \frac{g_{1}}{\|g_{1}\|}$; т.е. при $k=1$ - верно
2. Рассмотрим $g_{k+1}=1\cdot f_{k+1}-\underbrace{\lambda e_{1}-\dots-\lambda_{k}e_{k}}_{\in L(f_{1},\dots f_{k})}$
	$f_{1},\dots,f_{k},f_{k+1}$ - ЛН при $\forall\lambda_{1},\dots,\lambda_{k} \ g_{k+1}\neq \theta$
	Построим $g_{k+1}\perp e_{i}; \ i=\overline{1,k}$
	$$0=(g_{k+1},e_{i})=\left( f_{k+1}-\sum_{j=1}^{k}\lambda_{j}e_{j}, e_{i} \right)=(f_{k+1},e_{i})-\sum_{j=1}^{k}\lambda_{i}(e_{j},e_{i})=(f_{k+1},e_{i})-\lambda_{i}(e_{i},e_{i})\implies$$
	$$\implies\lambda_{i}=(f_{k+1},e_{i}); \ i=\overline{1,k}$$
	$$e_{k+1}= \frac{g_{k+1}}{\|g_{k+1}\|(\neq 0)}$$
	Построим ОНС $e_{1},\dots,e_{k}$
	Eсли $k+1=n$, то получим ОНБ, т.е. ОНБ $\exists$

$f_{1},\dots,f_{n}$ - базис $U$ $\to e_{1},\dots ,e_{n}$ - ОНБ $U$
$$1. g_{1}=f_{1};e_{1} = \frac{g_{1}}{\|g_{1}\|}$$
$$2. g_{2}=f_{2}-(f_{2},e_{1})e_{1}$$
$$\dots$$
$$k. \ \  g_{k}=f_{k}-\sum_{i=1}^{k-1}(f_{k},e_{i})e_{i}; \ \ e_{k}= \frac{g_{k}}{\|g_{k}\|} 
.$$$$\dots$$$$
n. \ \ g_{n}=f_{n}-\sum_{i=1}^{n-1}(f_{n},e_{i})e_{i};\ e_{n}= \frac{g_{n}}{\lVert g_{n} \rVert } $$
$\blacktriangleright$

**Замечание:** $[e]$ - ОНБ $U(E)$
$$x=[e]x_{\downarrow}\ \ \ \ x_{i}=?$$ $$x=\sum_{i=1}^{n}x_{i}e_{i} \ \bigg | (*,e_{j})$$
$$(x,e_{j})=\sum_{i=1}^{n}x_{i}(e_{i},e_{j})=x_{j}\cdot1; \ \ j=\overline{1,\dots n}$$
