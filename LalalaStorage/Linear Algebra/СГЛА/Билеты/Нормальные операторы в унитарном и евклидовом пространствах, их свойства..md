$A\in L(U,U); A^{*}A=A A^{*}$
1. $\forall x,y\in U \ (Ax,Ay)=(A^{*}x,A^{*}y)$
$\blacktriangleleft$
$(Ax,Ay)=(x,A^{*}(Ay))=(x,(A^{*} A)y)=(x,(A A^{*})y)=(x,A(A^{*}y))=\overline{(A(A^{*}y),y)}=\overline{(A^{*}y,A^{*}x)}=\overline{\overline{(A^{*}x,A^{*}y)}}=(A^{*}x,A^{*}y)$
$\blacktriangleright$
**Следствие:** $(Ax,Ax)=(A^{*}x,A^{*}x)\implies\|Ax\|=\|A^{*}x\|$
2. $\forall\lambda \in \mathbb{C}(\mathbb{R}) \ \ A-\lambda I$ - нормальный.
$\blacktriangleleft$
$(A-\lambda I)^{*}(A-\lambda I)=(A^{*}- \overline{\lambda}\underbrace{{I^{*}}}_{=I})(A-\lambda I)=(A^{*}-\overline{\lambda}I)A+(A^{*}-\overline{\lambda}I)(-\lambda I)=$
$\underbrace{A^{*}A}_{=A A^{*}}+\underbrace{(\overline{\lambda}I)A}_{-\overline{\lambda}IA}+\underbrace{A^{*}(-\lambda I)}_{=-\lambda A^{*}I}+(-\overline{\lambda}I)(-\lambda I)=A A^{*}+A(-\overline{\lambda}I)+(-\lambda I)A^{*}+(-\lambda)(-\overline{\lambda})II=$
$(A-\lambda I)A^{*}+(A-\lambda I)(-\overline{\lambda}I)=(A-\lambda I)(A^{*}-\overline{\lambda}I)=(A-\lambda I)(A-\lambda I)^{*}$ 
$\blacktriangleright$
3. $Ax=\lambda x, \ x\neq\theta\implies \ A^{*}x=\overline{\lambda}x$
$\blacktriangleleft$
$Ax=\lambda x\Leftrightarrow (A-\lambda I)x=\theta\Leftrightarrow\|(A-\lambda I)x\|=0\implies\|(A-\lambda I)^{*}x\|=0$ по следствию из св.1
$\Leftrightarrow(A^{*}-\overline{\lambda}I)x=\theta\Leftrightarrow A^{*}x=\overline{\lambda}x;x\neq\theta$
$\blacktriangleright$

4. $Ax_{i}=\lambda_{i} x_{i};\ i=\overline{1,2},\ x_{i}\neq\theta$ Если $\lambda_{1}\neq\lambda_{2}\implies x_{1}\perp x_{2}$
$\blacktriangleleft$
$(Ax_{1},x_{2})=(\lambda_{1}x_{1},x_{2})=\lambda_{1}(x_{1},x_{2})$
$(Ax_{1},x_{2})=(x_{1},A^{*}x_{2})=(x_{1},\overline{\lambda_{2}}x_{2})=\lambda_{2}(x_{1},x_{2})$
$0=\lambda_{1}(x_{1},x_{2})-\lambda_{2}(x_{1},x_{2})=(\lambda_{1}-\lambda_{2})(x_{1},x_{2})\implies(x_{1},x_{2})=0\implies x_{1}\perp x_{2}$
$\blacktriangleright$
