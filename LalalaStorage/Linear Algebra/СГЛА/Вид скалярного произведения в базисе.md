$[a]=(a_{1},\dots,a_{n})$ - базис $U(E)$.
$x=[a]\cdot x_{\downarrow}=\sum_{i=1}^{n}x_{i}a_{i}$ 
$y=[a]\cdot y_{\downarrow}=\sum_{j=1}^{n}y_{j}\cdot a_{j}$
$$(x,y)=\left( \sum_{i=1}^{n}x_{i}\cdot a_{i},\sum_{j=1}^{n}y_{j}\cdot a_{j} \right)=\sum_{i=1}^{n}x_{i}\left( a_{i}, \sum_{j=1}^{n}y_{j}\cdot a_{j} \right)=\sum_{i=1}^{n}x_{i}\sum_{j=1}^{n}\overline{y_{j}}(a_{i},a_{j})=$$
$$=\sum_{i=1}^{n}\sum_{j=1}^{n}(a_{i},a_{j})\cdot x_{i}\cdot \overline{y_{j}}=\sum_{i,j=1}^{n}g_{ij}\cdot x_{i}\cdot \overline{y_{j}}=$$ $g_{ij}=(a_{i},a_{j}), \ i,j=\overline{1,n}$ 
**Определение:** 
$$G_{a}=\|g_{ij}\|_{n\times n}=\|(a_{i},a_{j})\|_{n\times n}$$
называется *матрицей Грама* базиса $a$.

$$\sum_{i=1}^{n}x_{i}[G_{a}\overline{y_{\downarrow}}]_{i}=x_{\downarrow}^{T}G_{a}\cdot \overline{y_{j}}$$
В $U:$ $(x,y)=x^{T}_{\downarrow}G_{a}\overline{y}$
В $E: \ \ (x,y)=x^{T}_{\downarrow}G_{a}y_{\downarrow}$ 

**Теорема 1 (Переход матрицы Грама от одного базиса к другому):

$[a]\to^{T}[\tilde{a}]:$
$$x=[\tilde{a}]\cdot \tilde{x}_{\downarrow} \ \ \ \ x_{\downarrow}=T \tilde{x}_{\downarrow}$$
$$y=[\tilde{a}] \tilde{y}_{\downarrow} \ \ \ \ y_{\downarrow}=T \tilde{y}_{\downarrow}$$
Тогда $(1)\implies(x,y)= \tilde{x}_{\downarrow}^{T}G_{\tilde{a}} \overline{\tilde{y}}_{\downarrow}$
$$(1)\implies(x,y)=x_{\downarrow}^{T}G_{a} \overline{y}_{\downarrow} = (T \tilde{x}_{\downarrow})^{T}G_{a} \overline{(T \tilde{y}_{\downarrow})}=(\tilde{x}_{\downarrow})^{T}\cdot T^{T}G_{a}\cdot \overline{T} \overline{\tilde{y}}_{\downarrow}=$$
$$= \tilde{x}_{\downarrow}^{T}(T^{T}G_{a}\overline{T})(=B)\overline{\tilde{y}}_{\downarrow}$$
$\forall x,y\in U(E):$
$$\tilde{x}^{T}B \overline{\tilde{y}}= \tilde{x}^{T}G_{\tilde{a}} \overline{\tilde{y}}_{\downarrow} \ \ \ \overline{\tilde{y}}_{\downarrow}\in \mathbb{C}^{n}$$
$$\implies \tilde{x}^{T}B=\tilde{x}^{T}G_{\tilde{a}} \ \forall x_{\downarrow}\in \mathbb{C}^{n}\implies B=G_{\tilde{a}}$$
$$G_{\tilde{a}}=T^{T}G_{a}\overline{T} \ \text{в }U$$
$$G_{\tilde{a}}=T^{T}G_{a}T \ \text{в }E$$
Для $G_{a}: \ \ g_{ij}=(a_{i},a_{j})= \overline{(a_{j},a_{i})}=\overline{g_{ji}}, \ \ i,j=\overline{1,n}$
$G_{a}=(\overline{G_{a}})^{T}$ - эрмитова матрица в $U$
$G_{a}=(G_{a})^{T}$ - симметричная в $E$
