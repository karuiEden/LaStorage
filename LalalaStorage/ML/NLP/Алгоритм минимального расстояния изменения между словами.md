Пусть у нас даны две строки: $X$ длины $n$ и строка $Y$ длины $m$. Расстоянием $D[i,j]$ будет расстоянием изменения строк между $X[1..i]$ и $Y[1..j]$. Расстояние изменения между всеми полными строками обозначается $D[n,m]$.

Алгоритм является динамическим программированием, поэтому
$$
D[i,j] = \min\begin{cases}
D[i-1,j]+\text{del-cost(source[i])} \\
D[i,j-1]+\text{ins-cost(target[j])} \\
D[i-1,j-1]+\text{sub-cost(source[i], target[j])}
\end{cases}
$$
Схема алгоритма:
```pseudo
\begin{algorithm}
\caption{Min-Edit-Distance($source$, $target$)}
\begin{algorithmic}
\Require strings $source$, $target$
\Ensure minimum edit distance

\State $n \gets \text{LENGTH}(source)$
\State $m \gets \text{LENGTH}(target)$
\State Create distance matrix $D[n+1, m+1]$

\Comment{Initialization: distance from empty string}
\State $D[0,0] \gets 0$
\For{$i = 1$ \textbf{to} $n$}
  \State $D[i,0] \gets D[i-1,0] + del\text{-}cost(source[i])$
\EndFor
\For{$j = 1$ \textbf{to} $m$}
  \State $D[0,j] \gets D[0,j-1] + ins\text{-}cost(target[j])$
\EndFor

\Comment{Recurrence relation}
\For{$i = 1$ \textbf{to} $n$}
  \For{$j = 1$ \textbf{to} $m$}
    \State $D[i,j] \gets \min \Big(
      D[i-1,j] + del\text{-}cost(source[i]),
      D[i-1,j-1] + sub\text{-}cost(source[i], target[j]),
      D[i,j-1] + ins\text{-}cost(target[j])
    \Big)$
  \EndFor
\EndFor

\Comment{Termination}
\State \Return $D[n,m]$
\end{algorithmic}
\end{algorithm}
```
