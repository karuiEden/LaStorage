1. Система должна быть способной к обучению;
2. Паттерны обучаемые, они должны быть сложными;
3. Существующие данные: доступные данные или данные, которые возможно собрать;
4. Решаемая задача решается с помощью предсказаний;
5. Данные из будущего следуют тем же паттернам, что и данные для обучения.

![[Pasted image 20260208193153.png]]

### Дополнительные требования

6. Задача повторяема(т.е. повторяется один и тот же паттерн)
7. Цена неверного предсказания невелика
8. Масштабируемость 
9. Паттерны постоянно меняются

Также ML системы не должны быть
- Неэтичными
- Иметь более простое решение;
- Нерентабельными.
