### MCP

**Определение:** *MCP* - стандарт для инструментов агента.

Представляет собой клиент серверную архитектуру с 3 ролями
![[Pasted image 20260406185931.png]]


**Роли:**
1. Хост:
- Оркестрация LLM;
- Управление соединениями;
- Управляет UI;
- Управляет правами доступа;
- Разрешение на использование работы с данными юзера
2. Сервер:
- Регистрирует способности
- Обрабатывает запросы;
- Создает контекст;
- Обработка подключения клиента;
- Отправляет оповещения клиенту
3. Клиент:
- Общение по протоколу;
- Проверка возможностей;
- Запуск инструментов;
- Обновления в реальном времени.
