## Изучение данных

**Просмотр частей таблицы:**

```python
df.head(n) # Просмотр n первых строк таблицы

dfd.sample(n) # Просмотр n рандомных строк

df.tail(n) # Просмотр n последних строк таблицы

df.loc[i:j] # Слайс строк

df.loc[[1, 7 ,2]] # Определённые строки

df.columns # Выводит название всех колонок

df.columns["column"] # Столбец с именем "column"

df.loc[n, 'column'] # Элемент n строки столбца "column"

# Доступны слайсы, показанные выше, которые доступны как для строк, так и для столбцов

df.loc[df.column == n] # Выводит те строки, в которых значение column равно n

# Для более сложных условий используются логические операторы 

```

**Изменение данных в таблице:**

```python

df.loc[(df.column == n), column] = m # Меняет значения у строк со столбцом column со значением n на значение m
```

**Получение общей информации:**

```python
df.info 
```

Данный метод показывает общие данные о таблице: количество строк, столбцов, а также количество строк с ненулевым значением каждого столбца.
Помимо этого указывает тип данных столбцов.

```python
df.describe()
```

Метод `describe` выводит среднюю статистику числовых столбцов всей таблице, что помогает провести анализ данных перед обучением таблицы. 

## Обработка данных

**Просмотр пустых данных:**

```python

df.isna() 
# Отрисовывает таблицу с данными, где в качестве значений используется true(ячейка равна NaN) и false(в ином случае)

```

Также пустые ячейки можно суммировать:

```python

df.isna().sum(axis=0) # Суммирует по оси столбцов

df.isna().sum(axis=1) # Суммирует по оси строк

```

**Поиск уникальных значений**

Показывает, сколько различных значений встречается в столбце:

```python

df['Column'].values_counts()

```

**Нахождение моды**

```python

df['Column'].mode()

```


**Удаление пустых значений**

```python

df['Column'].dropna() # Удаление в определённом столбце

df.dropna(inplace=True) # Удаление во всей таблице

```

**Удаление значений**

```python

df.drop(columns=["col1", "col2"], inplace=True)


```

**Cоздание столбца:**

```python

df['col'] = ...
```
