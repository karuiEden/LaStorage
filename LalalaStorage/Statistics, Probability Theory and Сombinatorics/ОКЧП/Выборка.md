### Упорядоченные выборки.
**Перестановка** - это способ последовательного расположения объектов с учётом их порядка.
Перестановку $n$ объектов ещё называют **перестановкой длины $n$** . Количество перестановок обозначается $P_n$ 

$$P_n = n!$$ 


**Размещение** - упорядоченный набор из $k$ различных элементов, взятых из множества мощностью $n$, при этом $k \leq n$ . То есть некоторая перестановка $k$ выбранных элементов из $n$.
1. **Обозначение:**
$A_n^k$ - количество размещений из $n$ по $k$ без повторений.
$\bar {A_n^k}$ - количество размещений из $n$ по $k$ с повторениями.
2. **Счёт:**
$$A_{n}^{k}= \frac{n!}{(n-k)!}$$
*Док-во:* Используем правило произведения.
*Следствие:* $A^{0}_{n}=1$
*Док-во:*
$$A_{n}^{0}= \frac{n!}{(n-0)!}= \frac{n!}{n!}=1$$
*Следствие:* $A_{n}^{n}=P_{n}=n!$
*Док-во:* $$A_{n}^{n}= \frac{n!}{(n-n)!}= \frac{n!}{0!}=n! =P_{n}$$
*Следствие:* Если $k>n$, то $A_{n}^{k}=0$

### Размещения с повторениями

$$\overline{A}_{n}^{k}=n^{k}$$
*Док-во:* Используем правило произведения, где у каждого объекта есть $n$ способов для выборки, а всего объектов $k$:
$$\prod_{i=1}^{k}n=n^{k}$$
*Следтвие:* при размещениях с повторениями может быть такой случай $k>n$, так как при выборе одного объекта их количество не уменьшается.
## Неупорядоченные выборки

### Без повторений

**Сочетание** - неупорядоченный набор из $k$ различных элементов, взятых из множества мощностью $n$, при этом $k \leq n$. То есть набор из $k$ выбранных элементов из $n$, в котором порядок выбора не важен.
1. **Обозначение:**
$C_n^k$ - количество сочетаний из $n$ по $k$ без повторений.
$\bar{C_n^k}$ - количество сочетаний из $n$ по $k$ с повторениями.
$$C_{n}^{k}= \frac{n!}{k!(n-k)!}$$
*Док-во:*
Размещение без повторений равно $A^{k}_{n}$. Но тут нам неважен порядок, а $A_{k}^{k}=k!$, тогда $C_{n}^{k}$ меньше $A_{n}^{k}$ в $k!$ раз.
*Следствие 1:* Симметричность $C_{n}^{k}=C_{n}^{n-k}$
*Док-во:*
$$C_{n}^{k}= \frac{n!}{k!(n-k)!}= \frac{n!}{(n+k-n)!(n-k)!}=C_{n}^{n-k}$$
*Следствие 2:* $C_{n}^{0}=C_{n}^{n}=1$
*Док-во:*
$$C_{n}^{0}= \frac{n!}{0!(n-0)!}= \frac{n!}{n!}=1= \frac{n!}{n!(n-n)!}=C_{n}^{n}$$
*Следствие 3:* Если $k>n$, то $C_{n}^{k}=0$

Также сочетания являются биномиальными коэффициентами в биноме Ньютона.

### С повторениями
$$\bar{C}_{n}^{k}=C_{n+k-1}^{k}=C^{n-1}_{n+k-1}$$
