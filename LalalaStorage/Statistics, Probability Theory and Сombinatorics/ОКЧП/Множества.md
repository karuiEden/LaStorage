$x\in A$, $A$ - множество, $x$ - элемент множества, $\in$ - принадлежность

| $\varnothing$ - пустое множество - множество, не содержащее ни одного элемента.

| **Подмножество** $A\subset B$, если $\forall x  \ (x\in A\implies x\in B)$

|**Равенство множеств** $A=B$: Если $A$ и $B$ содержат одни и те же элементы.

|**Следствие**: из определения равенства множеств следует, что равенство не зависит от кратности и порядка элементов: $\{ a,b,c \}=\{ a,a,a,c,c,b \}$

Обозначение: конечное множество - $\{ a, b, c \}$
#### Свойства множеств
1. **Рефлексивность**: $A\subset A$
2. **Транзитивность**: $A\subset B,B\subset C\implies A\subset C$
3. **Антисимметричность**: $A\subset B,B\subset A\implies A=B$

#### Отличие $\subset$ и $\in$
Пример: $A=\{ a,\{ b \},c,\{ c \} \}$
Тогда $\{ a \}\subset A \ \ \ \ \{ a \}\not\in A$
	$\{ b \}\not\subset A \ \ \ \{ b \}\in A \ \ \ \{ \{ b \} \}\subset A$
	$\{ c \}\subset A \ \ \ \{ c \}\in A$
