**Определение**: Событие $\bar{A}$ противоположно событию $A$. Если оно содержит все не благоприятные исходы для события $A$. То есть является дополнением $A$ до множества $\Omega$
$$\bar{A}=\Omega \setminus A=\{ x:x \in\Omega \wedge x \not\in A \}$$
Свойства противоположных событий:
1. $A\cup \bar{A}=\Omega$
2. $A\cap \bar{A}=\varnothing$
3. $\bar{\Omega}=\varnothing$
4. $\bar{\varnothing}=\Omega$
### Правило суммы противоположных событий

$$P(A)+P(\bar{A})=\frac{|A|}{|\Omega|}+\frac{|\bar{A
|}}{|\Omega|}=\frac{|A|+|\bar{A}|}{|\Omega|}=\frac{|\Omega|}{|\Omega|}=1$$
Следовательно, $\large P(A)=P(\bar{A})$

### Несовместные события

Несовместные события похожи на противоположные, но несовместных может быть больше двух.
$$A_{i}\cap A_{j}=\varnothing, \ \ i\neq j$$
Противоположные события являются частным случаем несовместных событий.

**Определение:** **Полная группа событий** - набор несовместных событий, которые в объединении дают всё пространство исходов $\Omega$.

### Совместные события

**Определение:** События $A$ и $B$ называют **совместными**, если $A\cap B\neq\varnothing$
#### Правило суммы для несовместных событий

$$P(A\cup B)=\frac{|A\cup B|}{|\Omega|}=\frac{|A|+|B|}{|\Omega|}=\frac{|A|}{|\Omega|}+\frac{|B|}{|\Omega|}=P(A)+P(B)$$
#### Правило суммы для совместных событий

$$P(A\cup B)=P(A)+P(B)-P(A\cap B)$$
*Док-во:*
$$P(A\cup B)= \frac{|A\cup B|}{|\Omega|}=\frac{|A|+|B|-|A\cap B|}{|\Omega|}= \frac{|A|}{|\Omega|}+ \frac{|B|}{|\Omega|}- \frac{|A\cap B|}{|\Omega|}=P(A)+P(B)-{P(A\cap B)}$$

Правило суммы для трёх совместных событий:
$$P(A\cup B\cup C)=P(A)+P(B)+P(C)-P(A\cap B)-P(B\cap C)-P(A\cap C)+P(A\cap B+\cap C)$$
