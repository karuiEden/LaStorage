$$P(B|A)= \frac{P(A\cap B)}{P(A)}\text{ - вероятность события } B \text{ при условии события } A$$
Условная вероятность возможна при $P(A)\neq 0$

Также можно выразить условную вероятность через классическое определение вероятности:
$$P(B|A)= \frac{P(A\cap B)}{P(A)}= \frac{ \frac{|A\cap B|}{|\Omega|}}{\frac{|A|}{|\Omega|}}= \frac{|A \cap B|\cdot|\Omega|}{|A|\cdot|\Omega|}= \frac{|A\cap B|}{|A|}$$
Если заменить условие на $\Omega$, то получим обычную вероятность:
$$P(B|\Omega)= \frac{|B\cap\Omega|}{|\Omega|}= \frac{|B|}{|\Omega|}=P(B)$$
### Произведение вероятностей

Через формулу условной вероятности можно выразить вероятность пересечения событий:
$$P(A\cap B)=P(B|A)\cdot P(A)$$
Также есть события, которые не зависят друг от друга, такие события называют **независимыми**:
| Событие $A$ не зависит от события $B$, если $P(B|A)=P(A)$

При независимых событиях формула вероятности пересечения событий выглядит так:
$$P(B\cap C)=P(B)\cdot P(C)$$
### Свойства условной вероятности

1. Отношение условных вероятностей
$$\frac{P(B|A)}{P(A|B)}=\frac{P(B)}{P(A)}$$ значит, $P(A)>P(B)$ равносильно $P(A|B)>P(B|A)$

2. Если $A\subset B$, то $P(B|A)=1$
3. Если $B\subset A$, то $P(B|A)= \frac{P(B)}{P(A)}$
4. Свойство для несовместных событий:
$$\text{Если } B_{1}\cap B_{2}=\varnothing\text{, то} P(B_{1}\cup B_{2}|A)=P(B_{1}|A)+P(B_{2}|A)$$
5. Свойство для противоположных событий:
$$P(B|A)+P(\bar{B}|A)=1$$
	Работает только с событием, в случае с условием:
$$P(B|A)+P(B|\bar{A})\neq 1$$
