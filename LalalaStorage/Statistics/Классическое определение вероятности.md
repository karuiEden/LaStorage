**Определение**: Вероятностью $P(A)$ события $A$ называется отношение числа элементарных событий $m$, благоприятствующих событию $A$, к числу всех элементарных событий $n$

$\omega_1, \omega_2, \omega_3, ..., \omega_n$ - элементарные события
Условия элементарных событий:
- Ими исчерпывается все возможные исходы эксперимента
- Никакие 2 из них не могут случится одновременно
- Все исходы являются равновероятными
$$P(\omega_i) = \frac{1}{n}$$
## Событие и его вероятность

**Определение**: Событие $A$ - это некоторое множество элементарных событий. Событие, которое не содержит элементарных событий, называется **невозможным**. Событие, которое содержит все элементарные события, называется **достоверным**.
$$P(A) = \frac{|A|}{n}$$
$P(A)$ - вероятность события
$n$ - количество элементарных событий
$|A|$ - количество элементов в $A$

$$P(\varnothing) = 0$$
$$P(\{\omega_1, \omega_2, \omega_3, ..., \omega_n\}) = 1$$
## Свойства вероятности

Если у эксперимента $n$ элементарных событий, то количество событий в данном эксперименте $2^n$ 

Достоверное событие обозначается символом $\Omega$ 
Свойства:
- $P(\varnothing) = 0, \quad P(\Omega) = 1$
- Если $A \notin \{\varnothing, \Omega\}$, то $P(A) = (0, 1)$
- $P(A) = 1 - P(\bar{A})$ 
- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$
