**Непрямоугольные матрицы** - структура данных, представляющая собой вектор целочисленных векторов произвольной длины.

![[Pasted image 20241114185114.png]]

где $m$ - количество строк матрицы, а $n_{1},\dots,n_{m}$ - длина строк.

Имеет более скудное количество методов по сравнению с обычной матрицей, а именно:
1. Создание
2. Вывод в консоль
3. Удаление.

Сама матрица состоит из вектора "строк" и длины этого вектора:
```C
typedef struct matrix {  
    uint32_t len_rows;  
    lines* rows;  
} matrix;
```
А сама строка состоит из её длины и вектора целочисленного типа данных:
```C
typedef struct {  
    uint32_t len_lines;  
    int32_t* lines;  
} lines;
```

Создание матрицы происходит при помощи динамического выделения памяти на вектор строк, затем происходит выделение и заполнение строк.
