**Определение:** *Дерево* - множество вершин тип `Node`:
- Пустое;
- состоит из корневой вершины и совокупности, связанных с ней поддеревьев.
Каждое поддерево - дерево.

```c
struct Node{
	Info info;

	Node **ptrs;
};
```

## Двоичное дерево

**Определение:** *Двоичное дерево* - множество вершин тип `Node`:
- пустое;
- состоит из корневой вершины и двух связанных с ней поддеревьев - левого и правого.
Каждое поддерево - двоичное дерево.

```C
struct Node{
	Info info;
	Node* left;
	Node* right;
};
```

![[Pasted image 20250612172422.png]]

![[Pasted image 20250612172349.png]]

**Представление обычного дерева с помощью двоичного дерева:** 
- Каждый узел $x$ содержит два указателя.
- Поле $left$ узла $x$ содержит указатель на самый левый дочерний узел для данного узла.
- Поле $right$ узла $x$ содержит указатель на соседний дочерний узел для его родительского узла.

**Определение:** *Обход дерева* - операция, при которой все вершины дерева обрабатываются в некотором порядке, при чём:
- ни одна вершина не будет пропущена;
- не одна вершина не будет обработна дважды.
Типы обходов:
- Прямой - (Корень, левый, правый);
- Симметричный - (левый, корень, правый);
- Обратный(В глубину) - (левый, правый, корень).
