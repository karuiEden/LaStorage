**Определение:** *Очередь* - логическая, динамическая структура данных с двумя концами, т.е. работает по принципу FIFO(first in first out).

**Операции:** 
- Добавление элемента в конец очереди - $O(1)$;
- Удаление элемента из начала очереди - $O(1)$;
- Просмотр первого элемента очереди - $O(1)$;
- Просмотр пустоты и/или заполненности очереди - $O(1)$.
