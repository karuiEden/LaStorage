## Поиск
![[Pasted image 20250609175641.png]]

```c
int search(Table t, int key){
	for (int i = 0; i < n; ++i){
		if (t[i].key == key){
			return i;
		}
	}
	return -1;
}
```

## Удаление элемента

```c
int erase(Table t, int key){
	int i = search(t, key);
	if (i == -1){
		return Error;
	}
	t[i] = t[n];
	n--;
	return OK;
}
```

Особенность в том, что перенос последнего элемента на место удаляемого.
