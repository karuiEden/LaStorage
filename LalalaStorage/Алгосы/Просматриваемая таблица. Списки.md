## Вставка
```c
int insert(Table t, int key, Info* info){
	if (search(t, key) != NULL){
		return Err;
	}
	Node* node = node_new(key, info);
	node->next = t.head;
	t.head = node;
	return OK;
}
```

Вставка идёт в начало списка, в отличие от реализации на векторе.
## Поиск
```c
Node* search(Table t, int key){
	if (!t.head){
		return NULL;
	}
	Node* ptr = t.head;
	for (;ptr;ptr = ptr->next){
		if (ptr.key == key){
			return ptr;	
		}
	}
	return NULL;
}
```
Вместо индексов возвращается указатель на элемент.
## Удаление
![[Pasted image 20250609182138.png]]

```c
int erase(Table t, int key){
	if (!t.head){
		return -1;
	}
	if (t.head->key == key){
		t.head = t.head->next;
		return OK;
	}
	Node* head = t.head;
	while (head->next){
		if (head->next->key == key){
			x = head->next;
			head->next = x->next;
			delete_node(x);
			return OK;
		}
		head = head->next;
	}
	return -1;
}
```

Никакой элемент не заменяется, а просто удаляется элемент из списка.
