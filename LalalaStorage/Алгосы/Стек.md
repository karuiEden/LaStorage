**Определение:** *Стек* - логическая, динамическая структура данных с одним концом. То есть работает про принципу LIFO(Last in first out).
![[Pasted image 20250609122958.png]]
**Операции, поддерживаемые стеком:**
- Добавление элемента в вершину стека(`push`) - $O(1)$
- Удаление элемента из вершины стека(`pop`) - $O(1)$
- Возврат элемента из вершины стека без удаления - $O(1)$
- Проверка на пустоту - $O(1)$
- Если реализован на векторе: Проверка на заполненность - $O(1)$
