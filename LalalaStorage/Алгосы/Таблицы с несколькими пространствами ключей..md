**Определение:** *Таблица с несколькими пространствами ключей* - таблица типа пары из коллекции ключей-значения.
$U_{1}=\{ k_{1}^{1},k_{2}^{1},\dots,k_{n}^{1} \}$
$U_{2}=\{ k_{1}^{2},\dots,k_{n}^{2} \}$
...
$U_{n}=\{ k_{1}^{n},k_{2}^{n},\dots,k_{n}^{n} \}$

Элемент таблицы: $<k^{1},k^{2},\dots,k^{n},info>, \ k^{1}\in U_{1},k^{2}\in U_{2},\dots,k^{n}\in U_{n}$

## Поиск

- по значению ключа из любого пространства ключей
	$T_{i}$ - эффективность поиска по $i$-ому пространству ключей.
- по значениям ключей из всех пространств ключей *необходимо* гарантировать, что по всем значениям ключей найден один и тот же элемент.
## Вставка

- задаются все ключи;
- поиск по каждому ключу не успешен;
Оценка эффективности:
$$\sum T_{i}$$
, где $T_{i}$ - оценка неуспешного поиска.

## Удаление

Необходимо модифицировать все пространства ключей
1. Просмотр всех подпространств ключей:
	- есть информация о полном ключе удаляемого элемента:
	$$\sum find_{i}(k)$$
	- нет информации о полном ключе удаляемого элемента:
	$$\sum scan_{i}(k)$$
2. Реорганизация структуры таблицы.
