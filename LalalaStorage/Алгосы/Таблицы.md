**Определение:** *Таблица* - это абстрактный тип данных, представляющий собой коллекцию пар "ключ-значение", где каждый ключ является идентификатором, который используются для доступа к соответствующему значению.

Элемент таблицы:

| ключ | информация |
| ---- | ---------- |

**Операции:**
- Вставка элемента в таблицу;
- Удаление элемента в таблицу;
- Поиск;
**Классификация:**
- [[Просматриваемая таблица|Просматриваемая]];
- [[Упорядоченная таблица|Упорядоченная]];
- С вычисляемым входом:
	- произвольного доступа;
	- перемешанные(хэш).
Также бывают с повторениями и без повторений.
