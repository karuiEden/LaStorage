**Определение:** *Хеш-таблица, использующая перемешивание **сложением*** - частный случай хеш-таблицы с открытой адресацией, которая использует линейное пробирование.
 $h'(k)$ - обычная хэш-функция
 $h(k,i)= (h'(k) + i)\mod m$ 
 Правило изменения:
 $j=h'(k)$
 $j=(j+1)\mod m$
 Но можно использовать и другие  шаги $p>1$, но тогда $p$ и $m$ должны быть взаимно простыми числами.
## Поиск

```c
int search(Table t, int key){
	int j = h(k) % t.msize;
	int n = 0;
	while(t.data[j].busy && n < t.msize){
		if (t.data[j].busy && t.data[j].key == key){
			return j;
		}
		j = (j + p) % t.msize;
		n++;
	}
	return -1;
}
```

## Вставка

```c
int insert(Table t, int key, int info){
	int j = h(k) % t.msize, n = 0;
	while (t.data[j].busy && n < m){
		j = (j + p) % msize;
		n++;	
	}
	if (n < m){
		t.data[j].key = key;
		t.data[j].info = info;
		return 0;
	}
	return -1;
}
```
